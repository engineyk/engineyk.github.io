## 1. 植被渲染概述

### 1.1 植被渲染的核心挑战

```
┌─────────────────────────────────────────────────────────────┐
│              植被渲染核心挑战                                  │
│                                                             │
│  1. 数量巨大                                                 │
│     • 一个场景可能有数十万到数百万棵草/树                      │
│     • 每棵草 50-200 三角形，百万棵 = 5000万-2亿三角形         │
│                                                             │
│  2. 半透明与 Alpha Test                                     │
│     • 树叶/草使用 Alpha Test 模拟形状                        │
│     • Alpha Test 破坏 Early-Z / HSR 硬件优化                │
│     • 移动端 TBR GPU 上代价尤其高                            │
│                                                             │
│  3. 动画需求                                                 │
│     • 风吹摆动（全局风 + 湍流）                               │
│     • 玩家交互弯曲                                           │
│     • 不能用骨骼动画（太贵），必须顶点 Shader 实现             │
│                                                             │
│  4. 光照复杂                                                 │
│     • 次表面散射（Subsurface Scattering）                    │
│     • 双面光照（Two-Sided Lighting）                         │
│     • 自阴影 + 环境遮蔽                                      │
│                                                             │
│  5. 带宽与 Overdraw                                         │
│     • 大量重叠的半透明面片                                    │
│     • Overdraw 可达 3-8x                                    │
│     • 移动端带宽是最大瓶颈                                    │
└─────────────────────────────────────────────────────────────┘
```

## 2. GPU Instancing 与批量渲染

### 2.1 Instancing 原理

```
┌─────────────────────────────────────────────────────────────┐
│              GPU Instancing 原理                              │
│                                                             │
│  传统渲染 (1 Draw Call per instance):                       │
│    DC 1: Draw Grass Mesh (Transform A)                      │
│    DC 2: Draw Grass Mesh (Transform B)                      │
│    DC 3: Draw Grass Mesh (Transform C)                      │
│    ...                                                      │
│    DC N: Draw Grass Mesh (Transform N)                      │
│    → N 个 Draw Call，CPU 瓶颈                               │
│                                                             │
│  GPU Instancing (1 Draw Call for all):                      │
│    DC 1: DrawInstanced(Grass Mesh, N instances)             │
│    → 1 个 Draw Call，GPU 并行处理 N 个实例                   │
│    → 每个实例通过 InstanceID 读取自己的变换矩阵               │
│                                                             │
│  Indirect Draw (GPU Driven):                                │
│    GPU Culling → 写入 IndirectArgs Buffer                   │
│    DC 1: DrawInstancedIndirect(Grass Mesh, ArgsBuffer)      │
│    → 实例数量由 GPU 决定（剔除后的数量）                      │
│    → CPU 完全不知道画了多少个                                 │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 Instance Data Buffer 设计

```hlsl
// ═══════════════════════════════════════════
// Per-Instance Data 结构设计
// ═══════════════════════════════════════════

// 方案1: 完整 4x4 矩阵 (64 bytes/instance)
// 优点: 支持任意变换
// 缺点: 带宽大
struct InstanceData_Full
{
    float4x4 LocalToWorld;  // 64 bytes
};

// 方案2: Position + Rotation + Scale (32 bytes/instance)
// 优点: 带宽减半
// 缺点: 需要在 Shader 中重建矩阵
struct InstanceData_Compact
{
    float3 Position;    // 12 bytes
    float4 Rotation;    // 16 bytes (Quaternion)
    float  Scale;       // 4 bytes  (Uniform scale)
};

// 方案3: 极致压缩 (16 bytes/instance) - 适合草地
// 草地通常不需要旋转X/Z，只需要 Y 轴旋转
struct InstanceData_Grass
{
    float3 Position;    // 12 bytes
    uint   Packed;      // 4 bytes
    // Packed:
    //   [0:7]   = Scale (0-255 → 0.0-2.0)
    //   [8:15]  = RotationY (0-255 → 0-360°)
    //   [16:23] = ColorVariation (0-255)
    //   [24:31] = TypeIndex (0-255, 草的种类)
};

// Shader 中解包
float UnpackScale(uint packed)
{
    return ((packed & 0xFF) / 255.0) * 2.0;
}

float UnpackRotationY(uint packed)
{
    return (((packed >> 8) & 0xFF) / 255.0) * 2.0 * PI;
}

float3 UnpackColor(uint packed)
{
    float variation = ((packed >> 16) & 0xFF) / 255.0;
    // Use variation to lerp between base colors
    return lerp(float3(0.2, 0.5, 0.1), float3(0.4, 0.7, 0.2), variation);
}
```

### 2.3 Unity 中的 GPU Instancing 实现

```csharp
// ═══════════════════════════════════════════
// Unity: Graphics.DrawMeshInstanced
// ═══════════════════════════════════════════
public class GrassRenderer : MonoBehaviour
{
    [SerializeField] private Mesh grassMesh;
    [SerializeField] private Material grassMaterial;
    
    private Matrix4x4[][] batches;       // 每批最多 1023 个
    private MaterialPropertyBlock[] mpbs;
    
    private const int BATCH_SIZE = 1023; // Unity 限制
    
    void Start()
    {
        // Generate grass positions
        List<Matrix4x4> matrices = GenerateGrassPositions();
        
        // Split into batches of 1023
        int batchCount = Mathf.CeilToInt(matrices.Count / (float)BATCH_SIZE);
        batches = new Matrix4x4[batchCount][];
        mpbs = new MaterialPropertyBlock[batchCount];
        
        for (int i = 0; i < batchCount; i++)
        {
            int start = i * BATCH_SIZE;
            int count = Mathf.Min(BATCH_SIZE, matrices.Count - start);
            batches[i] = new Matrix4x4[count];
            matrices.CopyTo(start, batches[i], 0, count);
            
            mpbs[i] = new MaterialPropertyBlock();
            // Per-instance color variation
            Vector4[] colors = new Vector4[count];
            for (int j = 0; j < count; j++)
                colors[j] = Color.Lerp(Color.green, 
                    new Color(0.3f, 0.6f, 0.1f), Random.value);
            mpbs[i].SetVectorArray("_Color", colors);
        }
    }
    
    void Update()
    {
        for (int i = 0; i < batches.Length; i++)
        {
            Graphics.DrawMeshInstanced(
                grassMesh, 0, grassMaterial, 
                batches[i], batches[i].Length, mpbs[i],
                ShadowCastingMode.Off,  // Grass usually no shadow
                false,                   // No receive shadow
                gameObject.layer
            );
        }
    }
}

// ═══════════════════════════════════════════
// Unity: Graphics.DrawMeshInstancedIndirect
// GPU Driven - 更高效
// ═══════════════════════════════════════════
public class GrassRendererIndirect : MonoBehaviour
{
    [SerializeField] private Mesh grassMesh;
    [SerializeField] private Material grassMaterial;
    
    private ComputeBuffer argsBuffer;
    private ComputeBuffer instanceBuffer;
    private ComputeBuffer visibleBuffer;  // Culling result
    
    [SerializeField] private ComputeShader cullingShader;
    
    private int totalInstances;
    private Bounds renderBounds;
    
    struct GrassInstance
    {
        public Vector3 position;
        public uint packed; // scale, rotation, color
    }
    
    void Start()
    {
        // Generate instances
        GrassInstance[] instances = GenerateInstances();
        totalInstances = instances.Length;
        
        // Upload to GPU
        instanceBuffer = new ComputeBuffer(
            totalInstances, sizeof(float) * 3 + sizeof(uint));
        instanceBuffer.SetData(instances);
        
        // Visible buffer (after culling)
        visibleBuffer = new ComputeBuffer(
            totalInstances, sizeof(float) * 3 + sizeof(uint),
            ComputeBufferType.Append);
        
        // Indirect args: [indexCount, instanceCount, 
        //                  startIndex, baseVertex, startInstance]
        argsBuffer = new ComputeBuffer(
            1, 5 * sizeof(uint), ComputeBufferType.IndirectArguments);
        uint[] args = new uint[5]
        {
            grassMesh.GetIndexCount(0), 0, 
            grassMesh.GetIndexStart(0),
            grassMesh.GetBaseVertex(0), 0
        };
        argsBuffer.SetData(args);
        
        // Set material buffers
        grassMaterial.SetBuffer("_VisibleInstances", visibleBuffer);
        
        renderBounds = new Bounds(Vector3.zero, Vector3.one * 10000);
    }
    
    void Update()
    {
        // GPU Culling
        visibleBuffer.SetCounterValue(0);
        
        cullingShader.SetBuffer(0, "_AllInstances", instanceBuffer);
        cullingShader.SetBuffer(0, "_VisibleInstances", visibleBuffer);
        cullingShader.SetMatrix("_VPMatrix", 
            Camera.main.projectionMatrix * Camera.main.worldToCameraMatrix);
        cullingShader.SetVector("_CameraPos", 
            Camera.main.transform.position);
        cullingShader.SetFloat("_MaxDistance", 200f);
        
        int threadGroups = Mathf.CeilToInt(totalInstances / 64f);
        cullingShader.Dispatch(0, threadGroups, 1, 1);
        
        // Copy visible count to args buffer
        ComputeBuffer.CopyCount(visibleBuffer, argsBuffer, sizeof(uint));
        
        // Draw
        Graphics.DrawMeshInstancedIndirect(
            grassMesh, 0, grassMaterial, renderBounds, argsBuffer);
    }
    
    void OnDestroy()
    {
        argsBuffer?.Release();
        instanceBuffer?.Release();
        visibleBuffer?.Release();
    }
}
```

### 2.4 GPU Culling Compute Shader

```hlsl
// GrassCulling.compute
#pragma kernel CSMain

struct GrassInstance
{
    float3 position;
    uint packed;
};

StructuredBuffer<GrassInstance> _AllInstances;
AppendStructuredBuffer<GrassInstance> _VisibleInstances;

float4x4 _VPMatrix;
float3 _CameraPos;
float _MaxDistance;
float4 _FrustumPlanes[6];

// Frustum culling test
bool IsInFrustum(float3 worldPos, float radius)
{
    for (int i = 0; i < 6; i++)
    {
        float dist = dot(_FrustumPlanes[i].xyz, worldPos) 
                   + _FrustumPlanes[i].w;
        if (dist < -radius)
            return false;
    }
    return true;
}

// Distance culling with fade
bool IsInRange(float3 worldPos)
{
    float dist = distance(worldPos, _CameraPos);
    return dist < _MaxDistance;
}

// Orientation culling - skip grass facing away from camera
bool IsVisible(float3 worldPos)
{
    float3 toCamera = normalize(_CameraPos - worldPos);
    // Grass is always visible (billboard-like)
    return true;
}

// Screen-space size culling
bool IsLargeEnough(float3 worldPos, float scale)
{
    float dist = distance(worldPos, _CameraPos);
    float screenSize = scale / dist;  // Approximate screen coverage
    return screenSize > 0.001;  // Skip sub-pixel instances
}

[numthreads(64, 1, 1)]
void CSMain(uint3 id : SV_DispatchThreadID)
{
    uint index = id.x;
    
    // Bounds check
    uint count, stride;
    _AllInstances.GetDimensions(count, stride);
    if (index >= count)
        return;
    
    GrassInstance inst = _AllInstances[index];
    float scale = ((inst.packed & 0xFF) / 255.0) * 2.0;
    
    // Multi-level culling
    if (!IsInRange(inst.position))
        return;
    
    if (!IsInFrustum(inst.position, scale * 0.5))
        return;
    
    if (!IsLargeEnough(inst.position, scale))
        return;
    
    // Passed all tests - add to visible list
    _VisibleInstances.Append(inst);
}
```

## 3. LOD（Level of Detail）系统

### 3.1 植被 LOD 策略

```
┌─────────────────────────────────────────────────────────────┐

│                                                             │

│  ──────── ──────────── ──────────── ──────────── ──────────  │
```

### 3.2 Dithered LOD Transition

```hlsl
// ═══════════════════════════════════════════
// Dithered LOD Transition
// 使用抖动图案实现无缝 LOD 过渡
// ═══════════════════════════════════════════

// 4x4 Bayer Dither Matrix
static const float4x4 DitherMatrix = float4x4(
    0.0/16.0,  8.0/16.0,  2.0/16.0, 10.0/16.0,
    12.0/16.0, 4.0/16.0, 14.0/16.0,  6.0/16.0,
    3.0/16.0, 11.0/16.0,  1.0/16.0,  9.0/16.0,
    15.0/16.0, 7.0/16.0, 13.0/16.0,  5.0/16.0
);

void ApplyDitheredLODTransition(float2 screenPos, float lodFade)
{
    // lodFade: 0 = fully LOD_current, 1 = fully LOD_next
    // During transition: 0 < lodFade < 1
    
    uint2 pixel = uint2(screenPos) % 4;
    float threshold = DitherMatrix[pixel.x][pixel.y];
    
    // Discard pixels based on dither pattern
    // LOD_current: discard when threshold < lodFade
    // LOD_next:    discard when threshold >= lodFade
    clip(threshold - lodFade);
}

// In fragment shader
float4 FragmentShader(v2f i) : SV_Target
{
    // Apply LOD transition dithering
    ApplyDitheredLODTransition(i.screenPos.xy, _LODFade);
    
    // Normal shading...
    float4 color = SampleAlbedo(i.uv);
    // ...
    return color;
}
```

### 3.3 Billboard / Impostor

```hlsl
// ═══════════════════════════════════════════
// Billboard - 始终面向摄像机的面片
// 适用于远处的树木
// ═══════════════════════════════════════════

// Cylindrical Billboard (Y-axis aligned)
// Tree trunk stays vertical, only rotates around Y
float3 BillboardVertex(float3 localPos, float3 worldCenter, 
                        float3 cameraPos)
{
    float3 toCamera = normalize(cameraPos - worldCenter);
    toCamera.y = 0; // Keep vertical
    toCamera = normalize(toCamera);
    
    float3 right = cross(float3(0, 1, 0), toCamera);
    float3 up = float3(0, 1, 0);
    
    float3 worldPos = worldCenter 
                    + right * localPos.x 
                    + up * localPos.y;
    return worldPos;
}

// ═══════════════════════════════════════════
// Octahedral Impostor - 多角度预渲染
// 比 Billboard 更真实，支持不同观察角度
// ═══════════════════════════════════════════

// Impostor Atlas: 8x8 = 64 个角度的预渲染图
// 存储: Albedo + Normal + Depth (packed into atlas)

struct ImpostorData
{
    float4 albedo;
    float3 normal;
    float  depth;
};

// Octahedral mapping: direction → UV
float2 OctahedralMap(float3 dir)
{
    float3 absDir = abs(dir);
    float sum = absDir.x + absDir.y + absDir.z;
    float2 oct = dir.xz / sum;
    
    if (dir.y < 0)
    {
        oct = (1.0 - abs(oct.yx)) * sign(oct);
    }
    
    return oct * 0.5 + 0.5; // [0, 1]
}

// Sample impostor from atlas
ImpostorData SampleImpostor(float3 viewDir, float2 localUV, 
                             Texture2D impostorAtlas, int gridSize)
{
    float2 octUV = OctahedralMap(viewDir);
    
    // Find the 4 nearest grid cells for bilinear interpolation
    float2 gridPos = octUV * (gridSize - 1);
    int2 cell00 = int2(floor(gridPos));
    float2 frac = gridPos - cell00;
    
    // Sample 4 cells and blend
    // Each cell contains a rendered view of the tree
    float2 cellSize = 1.0 / gridSize;
    float2 atlasUV = (cell00 + localUV) * cellSize;
    
    ImpostorData data;
    data.albedo = impostorAtlas.Sample(sampler_linear, atlasUV);
    // ... sample normal and depth similarly
    
    return data;
}
```

## 4. 风场动画系统

### 4.1 风场模型

```
┌─────────────────────────────────────────────────────────────┐
│              风场分层模型                                     │
│                                                             │
│  最终位移 = 全局风 + 阵风 + 湍流 + 细节噪声                   │
│                                                             │
│  Layer 1: Global Wind (全局风)                              │
│    • 恒定方向和强度                                          │
│    • 影响所有植被                                            │
│    • 低频，大幅度                                            │
│                                                             │
│  Layer 2: Gust (阵风)                                       │
│    • 周期性强度变化                                          │
│    • 使用 Perlin Noise 或 Sin 波叠加                        │
│    • 中频，中幅度                                            │
│                                                             │
│  Layer 3: Turbulence (湍流)                                 │
│    • 局部随机扰动                                            │
│    • 使用 3D Noise Texture                                  │
│    • 高频，小幅度                                            │
│                                                             │
│  Layer 4: Detail (细节)                                     │
│    • 叶片/草尖的微小抖动                                     │
│    • 基于顶点位置的 Hash                                     │
│    • 最高频，最小幅度                                        │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 草地风动画 Shader

```hlsl
// ═══════════════════════════════════════════
// Grass Wind Animation - Vertex Shader
// ═══════════════════════════════════════════

// Wind parameters (from CPU)
float3 _WindDirection;      // Normalized wind direction
float  _WindStrength;       // 0-1 wind intensity
float  _WindFrequency;      // Wind oscillation speed
float  _GustStrength;       // Gust intensity
float  _GustFrequency;      // Gust speed
float  _TurbulenceStrength; // Turbulence intensity
float  _Time;

Texture2D _WindNoiseTexture; // Tileable noise texture
SamplerState sampler_WindNoiseTexture;

// Grass vertex weight: 0 at root, 1 at tip
// Stored in vertex color or UV
float GetWindWeight(float3 localPos, float grassHeight)
{
    // Quadratic falloff - root stays fixed, tip moves most
    float normalizedHeight = saturate(localPos.y / grassHeight);
    return normalizedHeight * normalizedHeight;
}

float3 CalculateWindOffset(float3 worldPos, float windWeight)
{
    // ─── Layer 1: Global Wind ───
    float globalPhase = dot(worldPos.xz, _WindDirection.xz) * 0.05
                      + _Time * _WindFrequency;
    float globalWave = sin(globalPhase) * 0.5 + 0.5;
    float3 globalWind = _WindDirection * globalWave * _WindStrength;
    
    // ─── Layer 2: Gust ───
    float gustPhase = _Time * _GustFrequency;
    float2 gustUV = worldPos.xz * 0.01 + float2(gustPhase, 0);
    float gustNoise = _WindNoiseTexture.SampleLevel(
        sampler_WindNoiseTexture, gustUV, 0).r;
    float gust = gustNoise * _GustStrength;
    float3 gustWind = _WindDirection * gust;
    
    // ─── Layer 3: Turbulence ───
    float2 turbUV = worldPos.xz * 0.05 
                  + float2(_Time * 0.3, _Time * 0.2);
    float2 turbNoise = _WindNoiseTexture.SampleLevel(
        sampler_WindNoiseTexture, turbUV, 0).rg;
    float3 turbulence = float3(
        turbNoise.x - 0.5, 0, turbNoise.y - 0.5
    ) * _TurbulenceStrength;
    
    // ─── Layer 4: Detail (per-vertex) ───
    float detailPhase = dot(worldPos, float3(1.37, 2.63, 0.89)) 
                      + _Time * 4.0;
    float detail = sin(detailPhase) * 0.02;
    float3 detailOffset = float3(detail, 0, detail * 0.7);
    
    // ─── Combine ───
    float3 totalWind = globalWind + gustWind + turbulence + detailOffset;
    
    // Apply weight (root stays fixed)
    totalWind *= windWeight;
    
    // Preserve length - bend instead of stretch
    // Use rotation instead of translation for more realistic bending
    return totalWind;
}

// Improved: Rotation-based bending (preserves grass length)
float3 ApplyWindBending(float3 localPos, float3 worldPos, 
                         float windWeight, float grassHeight)
{
    float3 windForce = CalculateWindOffset(worldPos, 1.0);
    float windMagnitude = length(windForce);
    
    if (windMagnitude < 0.001)
        return localPos;
    
    // Calculate bend angle (clamped to prevent over-bending)
    float bendAngle = windMagnitude * windWeight * 0.5; // radians
    bendAngle = min(bendAngle, PI * 0.4); // Max 72 degrees
    
    // Bend axis is perpendicular to wind direction and up
    float3 windDir = normalize(windForce);
    float3 bendAxis = normalize(cross(float3(0, 1, 0), windDir));
    
    // Rotate vertex around root
    float cosA = cos(bendAngle * windWeight);
    float sinA = sin(bendAngle * windWeight);
    
    // Rodrigues' rotation formula
    float3 rotated = localPos * cosA 
                   + cross(bendAxis, localPos) * sinA
                   + bendAxis * dot(bendAxis, localPos) * (1 - cosA);
    
    return rotated;
}

// Full vertex shader
v2f VertexShader(appdata v, uint instanceID : SV_InstanceID)
{
    v2f o;
    
    // Get instance data
    GrassInstance inst = _VisibleInstances[instanceID];
    float scale = UnpackScale(inst.packed);
    float rotY = UnpackRotationY(inst.packed);
    
    // Local space operations
    float3 localPos = v.vertex.xyz * scale;
    float windWeight = GetWindWeight(localPos, scale);
    
    // Apply wind bending in local space
    localPos = ApplyWindBending(localPos, inst.position, 
                                 windWeight, scale);
    
    // Apply Y rotation
    float cosR = cos(rotY);
    float sinR = sin(rotY);
    float3 rotatedPos = float3(
        localPos.x * cosR - localPos.z * sinR,
        localPos.y,
        localPos.x * sinR + localPos.z * cosR
    );
    
    // World position
    float3 worldPos = rotatedPos + inst.position;
    
    o.positionCS = mul(UNITY_MATRIX_VP, float4(worldPos, 1));
```

```hlsl
// ═══════════════════════════════════════════
// Tree Wind Animation - Multi-Level
// ═══════════════════════════════════════════

// Tree wind uses vertex color channels:
// R = Branch Phase (unique per branch)
// G = Branch Weight (0 at trunk, 1 at branch tip)
// B = Leaf Weight (0 at branch, 1 at leaf tip)
// A = Trunk Weight (0 at root, 1 at top)

float3 TreeWindAnimation(float3 localPos, float3 worldPos,
                          float4 vertexColor)
{
    float trunkWeight = vertexColor.a;
    float branchWeight = vertexColor.g;
    float leafWeight = vertexColor.b;
    float branchPhase = vertexColor.r;
    
    float3 offset = float3(0, 0, 0);
    
    // ─── Level 1: Trunk Sway ───
    // Slow, large movement of entire tree
    float trunkPhase = _Time * _WindFrequency * 0.5 
                     + worldPos.x * 0.01;
    float trunkSway = sin(trunkPhase) * _WindStrength * 0.3;
    offset += _WindDirection * trunkSway * trunkWeight;
    
    // ─── Level 2: Branch Movement ───
    // Medium speed, per-branch variation
    float branchPhaseOffset = branchPhase * PI * 2.0;
    float branchOsc = sin(_Time * _WindFrequency * 1.5 
                        + branchPhaseOffset) 
                    * _WindStrength * 0.5;
    offset += _WindDirection * branchOsc * branchWeight;
    
    // Add vertical bounce for branches
    float branchBounce = cos(_Time * _WindFrequency * 2.0 
                           + branchPhaseOffset) 
                       * _WindStrength * 0.15;
    offset.y += branchBounce * branchWeight;
    
    // ─── Level 3: Leaf Flutter ───
    // Fast, small, chaotic movement
    float leafPhase1 = dot(localPos, float3(1.7, 3.1, 2.3)) 
                     + _Time * 6.0;
    float leafPhase2 = dot(localPos, float3(2.3, 1.1, 3.7)) 
                     + _Time * 5.0;
    float3 leafFlutter = float3(
        sin(leafPhase1) * 0.03,
        cos(leafPhase2) * 0.02,
        sin(leafPhase1 + leafPhase2) * 0.03
    ) * _WindStrength * leafWeight;
    offset += leafFlutter;
    
    return offset;
}
```

```hlsl
              * _InteractionCenter.w + 0.5;
```

### 5.3 Shader Parameter Array 交互

```hlsl
// Simple approach: pass interactor positions as array
#define MAX_INTERACTORS 8

float4 _InteractorPositions[MAX_INTERACTORS]; // xyz = pos, w = radius
float4 _InteractorVelocities[MAX_INTERACTORS]; // xyz = velocity, w = strength
int _InteractorCount;

float3 CalculateInteraction(float3 worldPos, float weight)
{
    float3 totalBend = float3(0, 0, 0);
    
    for (int i = 0; i < _InteractorCount; i++)
    {
        float3 interactorPos = _InteractorPositions[i].xyz;
        float radius = _InteractorPositions[i].w;
        float strength = _InteractorVelocities[i].w;
        
        float3 diff = worldPos - interactorPos;
        float dist = length(diff.xz);
        
        if (dist > radius)
            continue;
        
        // Falloff: 1 at center, 0 at radius
        float falloff = 1.0 - saturate(dist / radius);
        falloff = falloff * falloff; // Quadratic falloff
        
        // Push direction (away from interactor)
        float3 pushDir = normalize(float3(diff.x, 0, diff.z));
        
        // Add velocity influence
        float3 velDir = normalize(
            _InteractorVelocities[i].xyz + float3(0.001, 0, 0));
        pushDir = lerp(pushDir, velDir, 0.3);
        
        totalBend += pushDir * falloff * strength;
    }
    
    // Flatten grass when heavily bent
    float bendStrength = length(totalBend);
    totalBend.y = -bendStrength * 0.5; // Push down
    
    return totalBend * weight;
}
```

### 6.1 次表面散射（Subsurface Scattering）

```hlsl
// ═══════════════════════════════════════════
// Foliage Subsurface Scattering
// Light passes through thin leaves
// ═══════════════════════════════════════════

float3 FoliageSSS(float3 worldNormal, float3 viewDir, 
                   float3 lightDir, float3 lightColor,
                   float thickness, float3 sssColor)
{
    // Wrap lighting for softer look
    float NdotL = dot(worldNormal, lightDir);
    float wrapDiffuse = saturate(NdotL * 0.5 + 0.5);
    
    // Subsurface scattering
    // Light that passes through the leaf
    float3 H = normalize(lightDir + worldNormal * 0.6); // Distortion
    float VdotH = saturate(dot(viewDir, -H));
    float sss = VdotH * VdotH; // Power
    sss = saturate(sss * thickness); // Modulate by thickness
    
    // Back lighting (view-dependent translucency)
    float backLight = saturate(dot(viewDir, -lightDir));
    backLight = pow(backLight, 3.0) * thickness;
    
    float3 result = lightColor * (
        wrapDiffuse * 0.7 +           // Wrapped diffuse
        sss * sssColor * 2.0 +         // Subsurface scattering
        backLight * sssColor * 0.5     // Back lighting
    );
    
    return result;
}
```

### 6.2 双面光照（Two-Sided Lighting）

```hlsl
// ═══════════════════════════════════════════
// Two-Sided Foliage Lighting
// Leaves are visible from both sides
// ═══════════════════════════════════════════

float3 TwoSidedFoliageLighting(float3 worldNormal, float3 viewDir,
                                float3 lightDir, float3 lightColor,
                                float3 albedo, bool isFrontFace)
{
    // Flip normal for back face
    float3 N = isFrontFace ? worldNormal : -worldNormal;
    
    // Front face: standard PBR
    float NdotL_front = saturate(dot(worldNormal, lightDir));
    float3 frontLighting = albedo * lightColor * NdotL_front;
    
    // Back face: translucent lighting
    float NdotL_back = saturate(dot(-worldNormal, lightDir));
    float3 backLighting = albedo * lightColor * NdotL_back * 0.5;
    
    // Subsurface contribution
    float3 sss = FoliageSSS(N, viewDir, lightDir, lightColor, 
                             0.5, albedo * float3(0.8, 1.0, 0.6));
    
    return frontLighting + backLighting + sss;
}

// Fragment shader
float4 FoliageFragment(v2f i, bool isFrontFace : SV_IsFrontFace) : SV_Target
{
    float4 albedo = tex2D(_MainTex, i.uv);
    
    // Alpha test
    clip(albedo.a - _Cutoff);
    
    float3 N = normalize(i.worldNormal);
    float3 V = normalize(_WorldSpaceCameraPos - i.worldPos);
    float3 L = normalize(_MainLightDirection);
    
    float3 color = TwoSidedFoliageLighting(
        N, V, L, _MainLightColor.rgb, albedo.rgb, isFrontFace);
    
    // Ambient
    float3 ambient = SampleSH(isFrontFace ? N : -N) * albedo.rgb;
    color += ambient;
    
    return float4(color, 1);
}
```

### 7.1 Alpha Test 的性能问题

```hlsl
// Avoids popping at LOD transitions
float HashedAlphaTest(float alpha, float3 worldPos)
{
    // Hash based on world position for stable pattern
    float hash = frac(sin(dot(worldPos.xz, float2(12.9898, 78.233))) * 43758.5453);
```

```csharp
// ═══════════════════════════════════════════
// Procedural Vegetation Placement
// ═══════════════════════════════════════════
public class VegetationPlacer : MonoBehaviour
{
    [System.Serializable]
    public struct VegetationRule
    {
        public Mesh mesh;
        public Material material;
        public float density;          // Instances per square meter
        public float minScale, maxScale;
        public float minSlope, maxSlope;  // Degrees
        public float minHeight, maxHeight; // World height
        public int[] allowedTerrainLayers; // Terrain texture indices
        public float noiseScale;          // Placement noise frequency
        public float noiseThreshold;      // Noise cutoff
    }
    
    [SerializeField] private VegetationRule[] rules;
    [SerializeField] private Terrain terrain;
    
    public List<Matrix4x4> GeneratePlacements(VegetationRule rule, 
                                               Bounds area)
    {
        var placements = new List<Matrix4x4>();
        TerrainData td = terrain.terrainData;
        
        // Poisson Disk Sampling for natural distribution
        float spacing = 1f / Mathf.Sqrt(rule.density);
        var poissonPoints = PoissonDiskSampling.Generate(
            area.size.x, area.size.z, spacing);
        
        foreach (var point in poissonPoints)
        {
            float worldX = area.min.x + point.x;
            float worldZ = area.min.z + point.y;
            
            // Get terrain info at this position
            Vector3 terrainLocal = new Vector3(
                (worldX - terrain.transform.position.x) / td.size.x,
                0,
                (worldZ - terrain.transform.position.z) / td.size.z
            );
            
            // Height check
            float height = terrain.SampleHeight(
                new Vector3(worldX, 0, worldZ));
            if (height < rule.minHeight || height > rule.maxHeight)
                continue;
            
            // Slope check
            float slope = td.GetSteepness(terrainLocal.x, terrainLocal.z);
            if (slope < rule.minSlope || slope > rule.maxSlope)
                continue;
            
            // Terrain layer check
            float[,,] alphamaps = td.GetAlphamaps(
                (int)(terrainLocal.x * td.alphamapWidth),
                (int)(terrainLocal.z * td.alphamapHeight), 1, 1);
            
            bool layerMatch = false;
            foreach (int layerIdx in rule.allowedTerrainLayers)
            {
                if (alphamaps[0, 0, layerIdx] > 0.3f)
                {
                    layerMatch = true;
                    break;
                }
            }
            if (!layerMatch) continue;
            
            // Noise check (for natural clustering)
            float noise = Mathf.PerlinNoise(
                worldX * rule.noiseScale, worldZ * rule.noiseScale);
            if (noise < rule.noiseThreshold)
                continue;
            
            // Create placement
            float scale = Random.Range(rule.minScale, rule.maxScale);
            float rotY = Random.Range(0f, 360f);
            Vector3 normal = td.GetInterpolatedNormal(
                terrainLocal.x, terrainLocal.z);
            
            // Align to terrain normal (optional)
            Quaternion rot = Quaternion.FromToRotation(Vector3.up, normal)
                           * Quaternion.Euler(0, rotY, 0);
            
            Vector3 pos = new Vector3(worldX, height, worldZ);
            placements.Add(
                Matrix4x4.TRS(pos, rot, Vector3.one * scale));
        }
        
        return placements;
    }
}

// Poisson Disk Sampling - Natural Distribution
// ═══════════════════════════════════════════
public static class PoissonDiskSampling
{
    public static List<Vector2> Generate(
        float width, float height, float minDist, int maxAttempts = 30)
    {
        float cellSize = minDist / Mathf.Sqrt(2);
        int gridW = Mathf.CeilToInt(width / cellSize);
        int gridH = Mathf.CeilToInt(height / cellSize);
        int[,] grid = new int[gridW, gridH];
        
        // Initialize grid to -1
        for (int x = 0; x < gridW; x++)
            for (int y = 0; y < gridH; y++)
                grid[x, y] = -1;
        
        var points = new List<Vector2>();
        var activeList = new List<int>();
        
        // Seed point
        Vector2 seed = new Vector2(width * 0.5f, height * 0.5f);
        AddPoint(seed, points, activeList, grid, cellSize);
        
        while (activeList.Count > 0)
        {
            int idx = Random.Range(0, activeList.Count);
            Vector2 point = points[activeList[idx]];
            bool found = false;
            
            for (int attempt = 0; attempt < maxAttempts; attempt++)
            {
                float angle = Random.Range(0f, Mathf.PI * 2f);
                float dist = Random.Range(minDist, minDist * 2f);
                Vector2 candidate = point + new Vector2(
                    Mathf.Cos(angle) * dist, Mathf.Sin(angle) * dist);
                
                if (candidate.x < 0 || candidate.x >= width ||
                    candidate.y < 0 || candidate.y >= height)
                    continue;
                
                if (!HasNeighbor(candidate, points, grid, 
                                  cellSize, minDist, gridW, gridH))
                {
                    AddPoint(candidate, points, activeList, grid, cellSize);
                    found = true;
                    break;
                }
            }
            
            if (!found)
                activeList.RemoveAt(idx);
        }
        
        return points;
    }
```

### 8.2 Density Map 驱动

```hlsl
// ═══════════════════════════════════════════
// Compute Shader: Generate instances from density map
// ═══════════════════════════════════════════
#pragma kernel GenerateInstances

Texture2D<float> _DensityMap;       // R = density (0-1)
Texture2D<float4> _TerrainHeightmap; // Height data
Texture2D<float4> _TerrainNormalmap; // Normal data

RWStructuredBuffer<GrassInstance> _OutputInstances;
RWStructuredBuffer<uint> _InstanceCount; // Atomic counter

float4 _TerrainBounds; // xy = min, zw = size
float _TerrainHeight;
float _BaseDensity;    // Instances per cell
uint _Seed;

// PCG hash for deterministic random
uint PCGHash(uint input)
{
    uint state = input * 747796405u + 2891336453u;
    uint word = ((state >> ((state >> 28u) + 4u)) ^ state) * 277803737u;
    return (word >> 22u) ^ word;
}

float RandomFloat(uint seed)
{
    return float(PCGHash(seed)) / 4294967295.0;
}

[numthreads(8, 8, 1)]
void GenerateInstances(uint3 id : SV_DispatchThreadID)
{
    uint2 texSize;
    _DensityMap.GetDimensions(texSize.x, texSize.y);
    
    if (id.x >= texSize.x || id.y >= texSize.y)
        return;
    
    float density = _DensityMap[id.xy];
    if (density < 0.01)
        return;
    
    // Number of instances in this cell
    uint instancesInCell = uint(density * _BaseDensity);
    
    // Fractional part: probabilistic extra instance
    float fractional = frac(density * _BaseDensity);
    uint cellSeed = id.x * 73856093u ^ id.y * 19349663u ^ _Seed;
    if (RandomFloat(cellSeed) < fractional)
        instancesInCell++;
    
    float2 cellSize = _TerrainBounds.zw / float2(texSize);
    float2 cellOrigin = _TerrainBounds.xy + float2(id.xy) * cellSize;
    
    for (uint i = 0; i < instancesInCell; i++)
    {
        uint seed = cellSeed + i * 12345u;
        
        // Random position within cell
        float2 localPos = float2(
            RandomFloat(seed),
            RandomFloat(seed + 1u)
        );
        float2 worldXZ = cellOrigin + localPos * cellSize;
        
        // Sample height
        float2 heightUV = (worldXZ - _TerrainBounds.xy) / _TerrainBounds.zw;
        float height = _TerrainHeightmap.SampleLevel(
            sampler_linear_clamp, heightUV, 0).r * _TerrainHeight;
        
        // Random scale and rotation
        float scale = lerp(0.8, 1.5, RandomFloat(seed + 2u));
        float rotY = RandomFloat(seed + 3u);
        float colorVar = RandomFloat(seed + 4u);
        
        // Pack instance data
        GrassInstance inst;
        inst.position = float3(worldXZ.x, height, worldXZ.y);
        inst.packed = uint(scale / 2.0 * 255) 
                    | (uint(rotY * 255) << 8)
                    | (uint(colorVar * 255) << 16);
        
        // Atomic append
        uint index;
        InterlockedAdd(_InstanceCount[0], 1, index);
        _OutputInstances[index] = inst;
    }
}
```

```hlsl
// Mobile Grass Optimization
// ═══════════════════════════════════════════

// 1. Use half precision everywhere
half4 MobileGrassFragment(v2f i) : SV_Target
{
    half4 albedo = tex2D(_MainTex, i.uv);
    
    // Alpha test with pre-multiplied alpha
    clip(albedo.a - _Cutoff);
    
    // Simple lighting (no PBR)
    half3 N = normalize(i.worldNormal);
    half NdotL = saturate(dot(N, _MainLightDir.xyz));
    
    // Wrap lighting for softer look
    half wrap = NdotL * 0.5h + 0.5h;
    
    // Simple SSS approximation
    half VdotL = saturate(dot(i.viewDir, -_MainLightDir.xyz));
    half sss = VdotL * VdotL * _SSSStrength;
    
    half3 color = albedo.rgb * i.color.rgb 
                * (_MainLightColor.rgb * wrap + _AmbientColor.rgb)
                + sss * _SSSColor.rgb;
    
    return half4(color, 1.0h);
}

// 2. Simplified wind (mobile)
half3 MobileWindOffset(float3 worldPos, half weight)
{
    // Single sin wave instead of noise texture
    half phase = dot(worldPos.xz, _WindDirection.xz) * 0.1h 
               + _Time.y * _WindFrequency;
    half wave = sin(phase);
    
    return _WindDirection * wave * _WindStrength * weight;
}
```

## 12. 面试高频问题

### 12.1 基础概念

**Q1: 植被渲染的主要性能瓶颈是什么？如何解决？**

> 三大瓶颈：
> 1. **Draw Call** - 数十万实例不能逐个绘制
>    → GPU Instancing / Indirect Draw，1个DC绘制所有同类植被
> 2. **三角形数量** - 百万级三角形
>    → LOD 系统 + Billboard/Impostor + GPU Culling 剔除不可见实例
> 3. **Alpha Test / Overdraw** - 大量半透明面片重叠
>    → 移动端用 Opaque 十字交叉 Mesh 替代 Alpha Test
>    → Alpha to Coverage (需要 MSAA)
>    → 控制草片层叠数量

**Q2: GPU Instancing 和 Indirect Draw 的区别？**

> - **GPU Instancing** (DrawMeshInstanced)：CPU 决定实例数量，传入 Transform 数组
>   - 优点：简单，兼容性好
>   - 缺点：CPU 需要做 Culling，有 1023 实例限制（Unity）
> - **Indirect Draw** (DrawMeshInstancedIndirect)：GPU 决定实例数量
>   - 优点：GPU Culling，无实例数限制，CPU 零开销
>   - 缺点：需要 Compute Shader，调试困难

**Q3: 为什么 Alpha Test 在移动端性能差？有什么替代方案？**

> Alpha Test (clip/discard) 在 Fragment Shader 中执行，GPU 无法提前知道哪些像素会被丢弃：
> - Mali: FPK 失效，无法提前杀死被遮挡片段
> - Adreno: LRZ 部分失效
> - Apple: HSR 失效，退化为传统前向渲染
> 
> 替代方案：
> 1. **Opaque 十字 Mesh** - 两个交叉的不透明四边形，无需 Alpha Test
> 2. **Alpha to Coverage** - 需要 MSAA，用采样覆盖率模拟透明
> 3. **Dithered Transparency** - 用抖动图案模拟，配合 TAA 效果好

### 12.2 进阶问题

**Q4: 如何实现自然的风吹效果？**

> 分层风场模型：
> 1. **全局风** - 恒定方向，Sin 波驱动，低频大幅度
> 2. **阵风** - Perlin Noise 调制强度，中频中幅度
> 3. **湍流** - 3D Noise Texture 采样，高频小幅度
> 4. **细节** - 基于顶点位置 Hash 的微小抖动
> 
> 关键技巧：
> - 用旋转（弯曲）而非平移，保持草的长度不变
> - 根部固定，顶部位移最大（二次权重曲线）
> - 限制最大弯曲角度（~72°），防止穿地

**Q5: 如何实现玩家与草地的交互？**

> 三种方案：
> 1. **Render Texture** - 从上方渲染交互体到 RT，草 Shader 采样 RT
>    - 优点：支持任意数量交互体，GPU 友好
>    - 缺点：精度受 RT 分辨率限制
> 2. **Shader 参数数组** - CPU 传递交互体位置数组
>    - 优点：精度高，简单
>    - 缺点：数量有限（<16）
> 3. **Compute Buffer** - 交互体数据存 StructuredBuffer
>    - 优点：灵活，支持复杂交互
>    - 缺点：需要 Compute Shader

**Q6: 如何处理大世界中的植被流式加载？**

> 1. **分块管理** - 世界划分为固定大小的 Chunk（如 64m×64m）
> 2. **按距离加载** - 玩家周围 N 个 Chunk 加载植被数据
> 3. **异步生成** - 植被实例在后台线程/Compute Shader 生成
> 4. **缓存复用** - 已生成的 Chunk 数据缓存，避免重复计算
> 5. **密度衰减** - 远处 Chunk 降低植被密度
> 6. **Instance Buffer 管理** - 使用环形缓冲区，避免频繁分配

**Q7: UE5 Nanite 对植被渲染有什么影响？**

> Nanite 可以用于树木等大型植被：
> - 自动 LOD，无需手动制作 LOD 链
> - 虚拟几何体，按需加载三角形
> - 硬件光栅 + 软件光栅混合
> 
> 限制：
> - 早期版本不支持 World Position Offset（无法做风动画）
> - UE 5.4+ 开始支持 Nanite + WPO
> - 不支持半透明和 Alpha Test（Masked 有限支持）
> - 草地通常仍用传统 Instancing（太小，Nanite 优势不明显）
