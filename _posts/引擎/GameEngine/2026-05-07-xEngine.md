## 三、渲染管线生命周期

### 3.1 应用启动流程 (main.cpp)

```cpp
int main(int argc, char** argv) {
    // 1. 初始化应用
    g_App->Init();
    
    // 2. 加载场景文件
    g_pSceneManager->LoadScene("Scene/test.ogex");
    
    // 3. 初始化图形管理器
    g_graphicsManager->Initialize();
    
    // 4. 主循环
    while(!g_App->IsQuit()) {
        g_App->Tick();  // 每帧执行
    }
    
    // 5. 清理
    g_App->OnDestroy();
}
```

### 3.2 帧循环 Tick() 调用链

```
OpenGLApp::Tick()
    │
    ├── WinApp::Tick()
    │   └── 处理 Windows 消息循环 (PeekMessage/DispatchMessage)
    │
    ├── g_graphicsManager->Tick()
    │   │
    │   ├── Clear()        ← 清空颜色/深度缓冲
    │   │   └── glClearColor() + glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT)
    │   │
    │   └── Draw()         ← 渲染场景
    │       └── OpenGLGraphicsManager::RenderScene()
    │
    └── SwapBuffers(m_hDC)  ← 交换前后缓冲区
```

### 3.3 OpenGLGraphicsManager::RenderScene() 详细流程

```cpp
void OpenGLGraphicsManager::RenderScene() {
    // Step 1: 更新世界矩阵 (旋转动画)
    rotateAngle += PI / 360;
    MatrixRotationZ(rotationMatrixZ, rotateAngle);
    drawFrameContext.worldMatrix = rotationMatrixZ;
    
    // Step 2: 计算相机矩阵
    CalculateCameraMatrix()
        └── 从场景获取相机节点
            ├── 计算 View Matrix (逆变换)
            └── 计算 Projection Matrix (透视投影)
    
    // Step 3: 计算光照
    CalculateLights()
        └── 从场景获取光照信息
    
    // Step 4: 设置逐帧 Shader 参数
    SetPerFrameShaderParameters()
        ├── glUseProgram(shaderProgram)
        ├── glUniformMatrix4fv("worldMatrix", ...)
        ├── glUniformMatrix4fv("viewMatrix", ...)
        ├── glUniformMatrix4fv("projectionMatrix", ...)
        └── glUniform3fv("lightPosition", ...) + glUniform4fv("lightColor", ...)
    
    // Step 5: 渲染所有几何体
    for (auto dbc : drawBatchContext) {
        SetPerBatchShaderParameters("modelMatrix", *dbc.transform)
        glBindVertexArray(dbc.vao)
        glMultiDrawElements(dbc.mode, dbc.counts.data(), dbc.type, ...)
    }
    
    glFlush()
}
```

## 四、底层调用原理详解

### 4.1 OpenGL 上下文创建 (OpenGLApp.cpp)

```
1. 创建临时窗口和上下文 (用于加载 WGL 扩展)
2. wglGetProcAddress() 加载 ARB 扩展函数
3. 创建正式窗口
4. wglChoosePixelFormatARB() 选择像素格式 (支持 MSAA)
5. wglCreateContextAttribsARB() 创建 Core Profile 4.3 上下文
6. gladLoadGL() 初始化 GLAD
```

```cpp
// 关键代码路径:
OpenGLApp::Init()
    → WinApp::CreateMainWindow()
    → wglChoosePixelFormatARB()  // 选择像素格式
    → wglCreateContextAttribsARB() // 创建 GL 4.3 上下文
    → BaseApp::Initialize()
    → gladLoadGL()              // 加载 GL 函数指针
```

### 4.2 资源加载流程

```
ResourcesManager::LoadText("Shaders/basic.vs")
    ↓
读取文件内容为 string
    ↓
OpenGLGraphicsManager::InitShader()
    ├── glCreateShader(GL_VERTEX_SHADER)
    ├── glShaderSource()
    ├── glCompileShader()
    ├── glCreateShader(GL_FRAGMENT_SHADER)
    ├── glShaderSource()
    ├── glCompileShader()
    ├── glCreateProgram()
    ├── glAttachShader()
    ├── glBindAttribLocation()  // 绑定顶点属性: inPosition(0), inNormal(1)
    └── glLinkProgram()
```

### 4.3 场景数据解析流程

```
OGEXSceneManager::LoadScene("Scene/test.ogex")
    ↓
ResourcesManager::LoadText() → 读取 .ogex 文件
    ↓
OgexParser::Parse() → 解析 OGEX 格式
    ↓
构建 Scene 对象:
    ├── SceneGraph (场景树根节点)
    ├── Cameras (unordered_map<string, SceneObjectCamera>)
    ├── Lights (unordered_map<string, SceneObjectLight>)
    ├── Geometries (unordered_map<string, SceneObjectGeometry>)
    └── GeometryNodes / CameraNodes / LightNodes (场景节点映射)
```

### 4.4 Buffer 初始化流程

```
OpenGLGraphicsManager::InitBuffer()
    ↓
遍历 Scene 中的所有 GeometryNode
    ↓
获取 Geometry → Mesh
    ↓
对每个 Mesh:
    ├── glGenVertexArrays()                         // 创建 VAO
    ├── glBindVertexArray()
    │
    ├── 遍历 VertexPropertyArray (位置、法线等):
    │   ├── glGenBuffers()                          // VBO
    │   ├── glBindBuffer(GL_ARRAY_BUFFER)
    │   ├── glBufferData()                          // 上传顶点数据
    │   ├── glEnableVertexAttribArray()
    │   └── glVertexAttribPointer()                 // 设置顶点属性格式
    │
    └── 遍历 IndexArray (每个子mesh):
        ├── glGenBuffers()                          // EBO
        ├── glBindBuffer(GL_ELEMENT_ARRAY_BUFFER)
        ├── glBufferData()                          // 上传索引数据
        └── 创建 DrawBatchContext 保存绘制参数
```

### 4.5 Shader Uniform 设置

```cpp
// 逐帧参数 (Per-Frame)
glUniformMatrix4fv(location("worldMatrix"), ...)        // 世界矩阵
glUniformMatrix4fv(location("viewMatrix"), ...)         // 视图矩阵  
glUniformMatrix4fv(location("projectionMatrix"), ...)   // 投影矩阵
glUniform3fv(location("lightPosition"), ...)            // 光照位置
glUniform4fv(location("lightColor"), ...)               // 光照颜色

// 逐批次参数 (Per-Batch)
glUniformMatrix4fv(location("modelMatrix"), ...)        // 模型变换矩阵
```

## 六、模块职责总结

| 模块          | 职责                                      | 关键类                            |
| ------------- | ----------------------------------------- | --------------------------------- |
| **Platform**  | Windows 窗口管理、消息循环、GL 上下文创建 | WinApp, OpenGLApp                 |
| **FrameWork** | 应用生命周期管理、场景管理、资源管理      | BaseApp, Scene, OGEXSceneManager  |
| **RHI**       | 图形 API 抽象层、渲染实现                 | OpenGLGraphicsManager             |
| **Graphics**  | 图形组件 (Mesh, Texture, Lighting 等)     | SceneObjectMesh, SceneObjectLight |
| **GfxDevice** | 底层图形设备抽象                          | GfxDevice                         |

## 七、渲染管线核心流程

```
应用启动
    ↓
创建窗口 / OpenGL 上下文
    ↓
加载场景 (.ogex 文件解析)
    ↓
初始化 Buffer (VAO, VBO, EBO)
    ↓
主循环开始
    ├── Clear()      → 清空颜色缓冲和深度缓冲
    ├── Draw()       → 执行渲染 (计算矩阵、设置 Uniform、绘制)
    └── SwapBuffers() → 交换前后缓冲区
    ↓
主循环结束 → 清理资源
```
