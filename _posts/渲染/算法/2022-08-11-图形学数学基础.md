## 1. 向量基础

### 1.1 向量表示

一个 n 维向量：

$$\vec{v} = (v_1, v_2, \ldots, v_n)$$

在图形学中最常用 2D、3D、4D 向量：

$$\vec{v}_{2D} = (x, y), \quad \vec{v}_{3D} = (x, y, z), \quad \vec{v}_{4D} = (x, y, z, w)$$

### 1.2 向量长度（模）

$$|\vec{v}| = \sqrt{x^2 + y^2 + z^2}$$

**推导**：基于勾股定理的 3D 扩展。

**性能优化**：

| 操作        | 指令开销      | 说明       |
| ----------- | ------------- | ---------- |
| `length(v)` | 高（含 sqrt） | 精确长度   |
| `dot(v, v)` | 低（仅乘加）  | 长度的平方 |

> **核心优化原则**：能用长度平方比较的场景，绝不求 `sqrt`。

```hlsl
// BAD: expensive sqrt
if (length(v) < radius)
    ...

// GOOD: compare squared values
if (dot(v, v) < radius * radius)
    ...
```

### 1.3 向量归一化

$$\hat{v} = \frac{\vec{v}}{|\vec{v}|} = \frac{\vec{v}}{\sqrt{\vec{v} \cdot \vec{v}}}$$

**快速逆平方根（Quake III 算法）**：

```c
float Q_rsqrt(float number) {
    long i;
    float x2, y;
    const float threehalfs = 1.5F;
    x2 = number * 0.5F;
    y  = number;
    i  = *(long*)&y;                       // evil floating point bit level hacking
    i  = 0x5f3759df - (i >> 1);            // what the fuck?
    y  = *(float*)&i;
    y  = y * (threehalfs - (x2 * y * y));  // 1st iteration of Newton's method
    // y  = y * (threehalfs - (x2 * y * y)); // 2nd iteration (optional)
    return y;
}
```

**推导原理**：
1. IEEE 754 浮点数 `f = (1 + m) × 2^e`，取对数后 `log(f) ≈ e + m`
2. 整数位操作 `i >> 1` 近似实现 `log(f) / 2`
3. 魔数 `0x5f3759df` 是对 `1/sqrt(x)` 初始猜测的最优常数
4. 一次牛顿迭代即可达到足够精度

> **现代 GPU**：直接使用硬件指令 `rsqrt(x)`，精度足够且只需 1 个时钟周期。

### 1.4 点积（Dot Product）

$$\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y + a_z b_z = |\vec{a}||\vec{b}|\cos\theta$$

**推导**（余弦定理法）：

由余弦定理：
$$|\vec{a} - \vec{b}|^2 = |\vec{a}|^2 + |\vec{b}|^2 - 2|\vec{a}||\vec{b}|\cos\theta$$

展开左边：
$$|\vec{a} - \vec{b}|^2 = (a_x - b_x)^2 + (a_y - b_y)^2 + (a_z - b_z)^2$$
$$= a_x^2 - 2a_xb_x + b_x^2 + a_y^2 - 2a_yb_y + b_y^2 + a_z^2 - 2a_zb_z + b_z^2$$
$$= |\vec{a}|^2 + |\vec{b}|^2 - 2(a_xb_x + a_yb_y + a_zb_z)$$

对比得：
$$|\vec{a}||\vec{b}|\cos\theta = a_xb_x + a_yb_y + a_zb_z$$

**图形学应用**：
- **光照计算**：`NdotL = dot(N, L)` — Lambert 漫反射
- **视角判断**：`NdotV = dot(N, V)` — 菲涅尔、边缘光
- **投影长度**：`proj = dot(v, normalize(axis))`
- **方向判断**：`dot > 0` 同向，`dot < 0` 反向，`dot == 0` 垂直

### 1.5 叉积（Cross Product）

$$\vec{a} \times \vec{b} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ a_x & a_y & a_z \\ b_x & b_y & b_z \end{vmatrix}$$

展开：
$$\vec{a} \times \vec{b} = (a_yb_z - a_zb_y, \; a_zb_x - a_xb_z, \; a_xb_y - a_yb_x)$$

**性质**：

- $|\vec{a} \times \vec{b}| = |\vec{a}||\vec{b}|\sin\theta$ — 平行四边形面积
- $\vec{a} \times \vec{b} = -\vec{b} \times \vec{a}$ — 反交换律
- 结果向量垂直于 $\vec{a}$ 和 $\vec{b}$ 所在平面

**图形学应用**：
- **法线计算**：三角形法线 `N = normalize(cross(e1, e2))`
- **TBN 矩阵**：切线空间构建
- **面朝向判断**：背面剔除
- **面积计算**：三角形面积 = `0.5 * length(cross(e1, e2))`

### 1.6 三重积

**标量三重积**（混合积）：
$$\vec{a} \cdot (\vec{b} \times \vec{c}) = \det \begin{pmatrix} a_x & a_y & a_z \\ b_x & b_y & b_z \\ c_x & c_y & c_z \end{pmatrix}$$

几何意义：三个向量构成的平行六面体体积（带符号）。

**向量三重积**：
$$\vec{a} \times (\vec{b} \times \vec{c}) = \vec{b}(\vec{a} \cdot \vec{c}) - \vec{c}(\vec{a} \cdot \vec{b})$$

> 记忆口诀：**BAC-CAB** 公式。

## 2. 矩阵与变换

### 2.1 基础矩阵运算

**矩阵乘法**（$C = AB$）：
$$C_{ij} = \sum_{k=1}^{n} A_{ik} B_{kj}$$

**性质**：
- 不满足交换律：$AB \neq BA$（一般情况）
- 满足结合律：$(AB)C = A(BC)$ — 这对变换合并至关重要
- $(AB)^{-1} = B^{-1}A^{-1}$
- $(AB)^T = B^T A^T$

### 2.2 平移矩阵

$$T(t_x, t_y, t_z) = \begin{pmatrix} 1 & 0 & 0 & t_x \\ 0 & 1 & 0 & t_y \\ 0 & 0 & 1 & t_z \\ 0 & 0 & 0 & 1 \end{pmatrix}$$

> **为什么需要 4×4 矩阵？** 3×3 矩阵只能表示线性变换（旋转、缩放、剪切），平移是仿射变换，需要齐次坐标将其统一为矩阵乘法。

**齐次坐标**：
- 点：$(x, y, z, 1)$ — w=1，受平移影响
- 向量：$(x, y, z, 0)$ — w=0，不受平移影响

### 2.3 缩放矩阵

**均匀缩放**：
$$S(s) = \begin{pmatrix} s & 0 & 0 & 0 \\ 0 & s & 0 & 0 \\ 0 & 0 & s & 0 \\ 0 & 0 & 0 & 1 \end{pmatrix}$$

**非均匀缩放**：
$$S(s_x, s_y, s_z) = \begin{pmatrix} s_x & 0 & 0 & 0 \\ 0 & s_y & 0 & 0 \\ 0 & 0 & s_z & 0 \\ 0 & 0 & 0 & 1 \end{pmatrix}$$

> **注意**：非均匀缩放会破坏法线方向！法线变换需要使用逆转置矩阵 $(M^{-1})^T$。

**法线变换推导**：

设切线向量 $\vec{t}$ 变换为 $\vec{t'} = M\vec{t}$，法线 $\vec{n}$ 变换为 $\vec{n'} = G\vec{n}$。

由正交性：$\vec{n'} \cdot \vec{t'} = 0$

$$(G\vec{n})^T (M\vec{t}) = \vec{n}^T G^T M \vec{t} = 0$$

要使上式对所有 $\vec{t}$ 成立，需要 $G^T M = I$，即：

$$G = (M^{-1})^T$$

### 2.4 旋转矩阵

**绕 X 轴旋转**：
$$R_x(\theta) = \begin{pmatrix} 1 & 0 & 0 \\ 0 & \cos\theta & -\sin\theta \\ 0 & \sin\theta & \cos\theta \end{pmatrix}$$

**绕 Y 轴旋转**：
$$R_y(\theta) = \begin{pmatrix} \cos\theta & 0 & \sin\theta \\ 0 & 1 & 0 \\ -\sin\theta & 0 & \cos\theta \end{pmatrix}$$

**绕 Z 轴旋转**：
$$R_z(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta & 0 \\ \sin\theta & \cos\theta & 0 \\ 0 & 0 & 1 \end{pmatrix}$$

**推导**（以绕 Z 轴为例）：

设点 $P = (r\cos\alpha, r\sin\alpha)$ 绕原点旋转 $\theta$ 角：

$$P' = (r\cos(\alpha + \theta), r\sin(\alpha + \theta))$$

展开三角函数：
$$x' = r\cos\alpha\cos\theta - r\sin\alpha\sin\theta = x\cos\theta - y\sin\theta$$
$$y' = r\cos\alpha\sin\theta + r\sin\alpha\cos\theta = x\sin\theta + y\cos\theta$$

写成矩阵形式即得 $R_z(\theta)$。

**旋转矩阵性质**：
- 正交矩阵：$R^T = R^{-1}$（逆等于转置，计算极快！）
- $\det(R) = 1$
- 列向量互相正交且为单位向量

**绕任意轴旋转（Rodrigues 公式）**：

绕单位轴 $\hat{n}$ 旋转 $\theta$ 角：

$$R(\hat{n}, \theta) = \cos\theta \cdot I + (1 - \cos\theta)(\hat{n} \otimes \hat{n}) + \sin\theta \cdot [\hat{n}]_\times$$

其中 $\hat{n} \otimes \hat{n}$ 是外积矩阵，$[\hat{n}]_\times$ 是反对称矩阵：

$$[\hat{n}]_\times = \begin{pmatrix} 0 & -n_z & n_y \\ n_z & 0 & -n_x \\ -n_y & n_x & 0 \end{pmatrix}$$

**推导**：

将向量 $\vec{v}$ 分解为平行于 $\hat{n}$ 和垂直于 $\hat{n}$ 的分量：

$$\vec{v}_{\parallel} = (\hat{n} \cdot \vec{v})\hat{n}$$
$$\vec{v}_{\perp} = \vec{v} - \vec{v}_{\parallel}$$

垂直分量在旋转平面内旋转：
$$\vec{v}_{\perp}' = \cos\theta \cdot \vec{v}_{\perp} + \sin\theta \cdot (\hat{n} \times \vec{v}_{\perp})$$

注意 $\hat{n} \times \vec{v}_{\perp} = \hat{n} \times \vec{v}$（因为 $\hat{n} \times \vec{v}_{\parallel} = 0$）

合并：
$$\vec{v}' = \vec{v}_{\parallel} + \vec{v}_{\perp}' = \cos\theta \cdot \vec{v} + (1 - \cos\theta)(\hat{n} \cdot \vec{v})\hat{n} + \sin\theta(\hat{n} \times \vec{v})$$

### 2.5 变换组合顺序

标准变换顺序（列向量右乘约定）：

$$M = T \cdot R \cdot S$$

即：先缩放 → 再旋转 → 最后平移。

> **性能优化**：预计算组合矩阵 `MVP = Projection × View × Model`，在 CPU 端完成，避免 GPU 逐顶点重复计算。

### 2.6 矩阵求逆优化

| 矩阵类型         | 求逆方法                     | 开销 |
| ---------------- | ---------------------------- | ---- |
| 正交矩阵（旋转） | 转置 $R^{-1} = R^T$          | 极低 |
| 均匀缩放+旋转    | $M^{-1} = \frac{1}{s^2} M^T$ | 低   |
| 平移矩阵         | 取反平移分量                 | 极低 |
| TRS 组合         | $M^{-1} = S^{-1} R^T T^{-1}$ | 中   |
| 一般 4×4         | 伴随矩阵法 / LU 分解         | 高   |

```hlsl
// Optimized inverse for orthogonal matrix (rotation only)
float3x3 InverseRotation(float3x3 R) {
    return transpose(R); // That's it!
}

// Inverse for TRS matrix (no shear)
float4x4 InverseTRS(float4x4 M) {
    float3 scale = float3(
        length(M[0].xyz),
        length(M[1].xyz),
        length(M[2].xyz)
    );
    float3x3 R = float3x3(
        M[0].xyz / scale.x,
        M[1].xyz / scale.y,
        M[2].xyz / scale.z
    );
    float3x3 invR = transpose(R);
    float3 invS = 1.0 / scale;
    float3 invT = -mul(invR, M[3].xyz) * invS;
    
    float4x4 result;
    result[0] = float4(invR[0] * invS.x, 0);
    result[1] = float4(invR[1] * invS.y, 0);
    result[2] = float4(invR[2] * invS.z, 0);
    result[3] = float4(invT, 1);
    return result;
}
```

## 3. 四元数

### 3.1 定义

$$q = w + xi + yj + zk = (w, \vec{v})$$

其中 $i^2 = j^2 = k^2 = ijk = -1$。

### 3.2 四元数乘法

$$q_1 q_2 = (w_1 w_2 - \vec{v}_1 \cdot \vec{v}_2, \; w_1\vec{v}_2 + w_2\vec{v}_1 + \vec{v}_1 \times \vec{v}_2)$$

**推导**：直接展开 $(w_1 + x_1i + y_1j + z_1k)(w_2 + x_2i + y_2j + z_2k)$，利用 $ij=k, jk=i, ki=j$ 等规则整理。

### 3.3 四元数表示旋转

绕单位轴 $\hat{n}$ 旋转 $\theta$ 角的四元数：

$$q = \left(\cos\frac{\theta}{2}, \; \sin\frac{\theta}{2} \cdot \hat{n}\right)$$

旋转向量 $\vec{v}$：

$$\vec{v}' = q \vec{v} q^{-1} = q \vec{v} q^*$$

其中 $q^* = (w, -\vec{v})$ 是共轭四元数（单位四元数的逆等于共轭）。

**为什么是半角？**

推导：将 $\vec{v}$ 视为纯四元数 $(0, \vec{v})$，展开 $q \vec{v} q^*$：

$$q \vec{v} q^* = (0, \; (w^2 - |\vec{u}|^2)\vec{v} + 2(\vec{u} \cdot \vec{v})\vec{u} + 2w(\vec{u} \times \vec{v}))$$

其中 $\vec{u} = \sin(\theta/2)\hat{n}$, $w = \cos(\theta/2)$。

代入三角恒等式：
- $w^2 - |\vec{u}|^2 = \cos^2(\theta/2) - \sin^2(\theta/2) = \cos\theta$
- $2w|\vec{u}| = 2\sin(\theta/2)\cos(\theta/2) = \sin\theta$
- $2|\vec{u}|^2 = 2\sin^2(\theta/2) = 1 - \cos\theta$

恰好得到 Rodrigues 公式！

### 3.4 四元数 vs 欧拉角 vs 旋转矩阵

| 特性     | 欧拉角   | 旋转矩阵            | 四元数             |
| -------- | -------- | ------------------- | ------------------ |
| 存储     | 3 float  | 9 float             | 4 float            |
| 万向锁   | 有       | 无                  | 无                 |
| 插值     | 差       | 差                  | 优（Slerp）        |
| 组合旋转 | 需转矩阵 | 矩阵乘法(27 mul)    | 四元数乘法(16 mul) |
| 归一化   | N/A      | 6 约束 Gram-Schmidt | 1 约束 normalize   |
| 直觉性   | 高       | 低                  | 低                 |

### 3.5 四元数转旋转矩阵

$$R = \begin{pmatrix} 1-2(y^2+z^2) & 2(xy-wz) & 2(xz+wy) \\ 2(xy+wz) & 1-2(x^2+z^2) & 2(yz-wx) \\ 2(xz-wy) & 2(yz+wx) & 1-2(x^2+y^2) \end{pmatrix}$$

**性能优化**：预计算中间值

```csharp
// Optimized quaternion to matrix conversion
float xx = q.x * q.x, yy = q.y * q.y, zz = q.z * q.z;
float xy = q.x * q.y, xz = q.x * q.z, yz = q.y * q.z;
float wx = q.w * q.x, wy = q.w * q.y, wz = q.w * q.z;
// Only 9 multiplications instead of 27
```

### 3.6 Slerp（球面线性插值）

$$\text{Slerp}(q_0, q_1, t) = \frac{\sin((1-t)\Omega)}{\sin\Omega} q_0 + \frac{\sin(t\Omega)}{\sin\Omega} q_1$$

其中 $\cos\Omega = q_0 \cdot q_1$。

**推导**：

在单位球面上，两点 $q_0, q_1$ 之间的大圆弧参数化。要求插值路径上角速度恒定。

设 $q(t) = a(t) q_0 + b(t) q_1$，由约束条件：
- $q(0) = q_0 \Rightarrow a(0) = 1, b(0) = 0$
- $q(1) = q_1 \Rightarrow a(1) = 0, b(1) = 1$
- $|q(t)| = 1$（保持单位长度）

解得上述 Slerp 公式。

**性能优化 — Nlerp 替代**：

```hlsl
// Slerp: expensive (2x sin, 1x acos)
float4 Slerp(float4 q0, float4 q1, float t) {
    float cosOmega = dot(q0, q1);
    if (cosOmega < 0) { q1 = -q1; cosOmega = -cosOmega; }
    float omega = acos(cosOmega);
    float sinOmega = sin(omega);
    float a = sin((1.0 - t) * omega) / sinOmega;
    float b = sin(t * omega) / sinOmega;
    return a * q0 + b * q1;
}

// Nlerp: cheap (just normalize), nearly identical for small angles
float4 Nlerp(float4 q0, float4 q1, float t) {
    if (dot(q0, q1) < 0) q1 = -q1;
    return normalize(lerp(q0, q1, t));
}
```

> **何时用 Nlerp？** 当两个四元数夹角 < 30° 时，Nlerp 与 Slerp 的误差 < 0.1%，且快 3-5 倍。动画混合中大量使用 Nlerp。

## 4. 三角函数与近似

### 4.1 基本恒等式

```
sin²θ + cos²θ = 1
sin(α ± β) = sinα cosβ ± cosα sinβ
cos(α ± β) = cosα cosβ ∓ sinα sinβ
tan(α ± β) = (tanα ± tanβ) / (1 ∓ tanα tanβ)

// Double angle
sin(2θ) = 2 sinθ cosθ
cos(2θ) = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ

// Half angle
sin²(θ/2) = (1 - cosθ) / 2
cos²(θ/2) = (1 + cosθ) / 2

// Product to sum
sinα sinβ = [cos(α-β) - cos(α+β)] / 2
cosα cosβ = [cos(α-β) + cos(α+β)] / 2
sinα cosβ = [sin(α+β) + sin(α-β)] / 2
```

### 4.2 小角度近似

当 $\theta \to 0$：

$$\sin\theta \approx \theta$$
$$\cos\theta \approx 1 - \frac{\theta^2}{2}$$
$$\tan\theta \approx \theta$$

**误差分析**：

| θ (rad) | θ (deg) | sin θ 误差 | cos θ 误差 |
| ------- | ------- | ---------- | ---------- |
| 0.01    | 0.57°   | 0.0017%    | 0.0000%    |
| 0.1     | 5.73°   | 0.17%      | 0.0042%    |
| 0.3     | 17.2°   | 1.5%       | 0.11%      |
| 0.5     | 28.6°   | 4.1%       | 0.52%      |

> **应用场景**：微小旋转增量、弹簧阻尼系统、相机抖动等。

### 4.3 快速三角函数近似

**多项式近似（Shader 友好）**：

```hlsl
// Fast sin approximation, max error ~0.001 for x in [-pi, pi]
// Based on Bhaskara I's approximation (7th century!)
float FastSin(float x) {
    // Normalize to [-pi, pi]
    x = x - floor(x / 6.28318530718 + 0.5) * 6.28318530718;
    
    // Parabolic approximation
    float y = x * (3.14159265359 - abs(x) * 1.27323954474);
    // Optional: one iteration for better precision
    y = 0.225 * (y * abs(y) - y) + y;
    return y;
}

// Minimax polynomial approximation (5th order)
// Max error: 1.7e-4
float FastSin5(float x) {
    float x2 = x * x;
    return x * (1.0 - x2 * (0.16666667 - x2 * 0.00833333));
}

// Fast atan2 approximation
// Max error: 0.07 radians (~4 degrees)
float FastAtan2(float y, float x) {
    float ax = abs(x), ay = abs(y);
    float mn = min(ax, ay), mx = max(ax, ay);
    float a = mn / mx;
    
    // Polynomial approximation of atan for [0,1]
    float s = a * a;
    float r = ((-0.0464964749 * s + 0.15931422) * s - 0.327622764) * s * a + a;
    
    if (ay > ax) r = 1.57079637 - r;
    if (x < 0) r = 3.14159274 - r;
    if (y < 0) r = -r;
    return r;
}
```

## 5. 插值方法

### 5.1 线性插值（Lerp）

$$\text{lerp}(a, b, t) = a + t(b - a) = (1-t)a + tb$$

> **注意**：两种写法数学等价，但 `a + t*(b-a)` 在浮点精度上更好（当 t=1 时精确返回 b）。

### 5.2 Hermite 插值（SmoothStep）

$$\text{smoothstep}(t) = 3t^2 - 2t^3$$

**推导**：Hermite 基函数，满足边界条件 $f(0)=0, f(1)=1, f'(0)=0, f'(1)=0$。

设 $f(t) = at^3 + bt^2 + ct + d$，代入四个条件解方程组。

**SmootherStep（Ken Perlin 改进）**：

$$\text{smootherstep}(t) = 6t^5 - 15t^4 + 10t^3$$

满足额外条件 $f''(0)=0, f''(1)=0$，二阶导数连续。

```hlsl
// Standard smoothstep
float smoothstep(float edge0, float edge1, float x) {
    float t = saturate((x - edge0) / (edge1 - edge0));
    return t * t * (3.0 - 2.0 * t);
}

// Perlin's smootherstep (C2 continuous)
float smootherstep(float edge0, float edge1, float x) {
    float t = saturate((x - edge0) / (edge1 - edge0));
    return t * t * t * (t * (t * 6.0 - 15.0) + 10.0);
}
```

### 5.3 贝塞尔曲线

**二次贝塞尔**（3 个控制点）：
$$B(t) = (1-t)^2 P_0 + 2t(1-t) P_1 + t^2 P_2$$

**三次贝塞尔**（4 个控制点）：
$$B(t) = (1-t)^3 P_0 + 3t(1-t)^2 P_1 + 3t^2(1-t) P_2 + t^3 P_3$$

**De Casteljau 算法**（数值稳定的递归求值）：

```hlsl
// De Casteljau for cubic Bezier - numerically stable
float3 CubicBezier(float3 P0, float3 P1, float3 P2, float3 P3, float t) {
    float3 A = lerp(P0, P1, t);
    float3 B = lerp(P1, P2, t);
    float3 C = lerp(P2, P3, t);
    float3 D = lerp(A, B, t);
    float3 E = lerp(B, C, t);
    return lerp(D, E, t);
}
```

### 5.4 Catmull-Rom 样条

经过所有控制点的插值样条：

$$q(t) = 0.5 \begin{pmatrix} 1 & t & t^2 & t^3 \end{pmatrix} \begin{pmatrix} 0 & 2 & 0 & 0 \\ -1 & 0 & 1 & 0 \\ 2 & -5 & 4 & -1 \\ -1 & 3 & -3 & 1 \end{pmatrix} \begin{pmatrix} P_{i-1} \\ P_i \\ P_{i+1} \\ P_{i+2} \end{pmatrix}$$

```hlsl
float3 CatmullRom(float3 P0, float3 P1, float3 P2, float3 P3, float t) {
    float t2 = t * t;
    float t3 = t2 * t;
    return 0.5 * (
        (2.0 * P1) +
        (-P0 + P2) * t +
        (2.0 * P0 - 5.0 * P1 + 4.0 * P2 - P3) * t2 +
        (-P0 + 3.0 * P1 - 3.0 * P2 + P3) * t3
    );
}
```

### 5.5 双线性插值

用于纹理采样：

$$f(x, y) = (1-s)(1-t) f_{00} + s(1-t) f_{10} + (1-s)t f_{01} + st f_{11}$$

其中 $s, t \in [0, 1]$ 是小数部分。

**等价写法**（两步 lerp）：

```hlsl
float BilinearLerp(float f00, float f10, float f01, float f11, float s, float t) {
    float a = lerp(f00, f10, s); // bottom edge
    float b = lerp(f01, f11, s); // top edge
    return lerp(a, b, t);        // vertical
}
```

### 6.2 Model 矩阵

$$M_{model} = T \cdot R \cdot S$$

将顶点从模型局部空间变换到世界空间。

### 6.3 View 矩阵

**推导**：View 矩阵是相机变换的逆。

设相机位置 $\vec{e}$，前方 $\hat{f}$，上方 $\hat{u}$，右方 $\hat{r}$：

$$V = \begin{pmatrix} r_x & r_y & r_z & -\vec{r} \cdot \vec{e} \\ u_x & u_y & u_z & -\vec{u} \cdot \vec{e} \\ -f_x & -f_y & -f_z & \vec{f} \cdot \vec{e} \\ 0 & 0 & 0 & 1 \end{pmatrix}$$

> 注意：OpenGL 使用右手坐标系，相机看向 $-Z$，所以前方取负。

**LookAt 构建**：

```hlsl
// Build view matrix from eye, target, up
float4x4 LookAt(float3 eye, float3 target, float3 worldUp) {
    float3 f = normalize(target - eye);         // forward
    float3 r = normalize(cross(f, worldUp));    // right
    float3 u = cross(r, f);                     // up (already normalized)
    
    return float4x4(
        r.x,  r.y,  r.z,  -dot(r, eye),
        u.x,  u.y,  u.z,  -dot(u, eye),
       -f.x, -f.y, -f.z,   dot(f, eye),
        0,    0,    0,     1
    );
}
```

### 6.4 透视投影矩阵

**推导**（OpenGL 风格，NDC $z \in [-1, 1]$）：

设视锥参数：近平面 $n$，远平面 $f$，垂直 FOV $\theta$，宽高比 $a$。

$$\text{top} = n \cdot \tan(\theta/2), \quad \text{right} = \text{top} \cdot a$$

透视投影的核心思想：将视锥体"压缩"为长方体（透视除法）。

对于视锥内一点 $(x, y, z)$（$z < 0$），投影到近平面：

$$x' = \frac{n \cdot x}{-z}, \quad y' = \frac{n \cdot y}{-z}$$

为了用矩阵表示除以 $z$，利用齐次坐标的 $w$ 分量：

$$P = \begin{pmatrix} \frac{n}{\text{right}} & 0 & 0 & 0 \\ 0 & \frac{n}{\text{top}} & 0 & 0 \\ 0 & 0 & -\frac{f+n}{f-n} & -\frac{2fn}{f-n} \\ 0 & 0 & -1 & 0 \end{pmatrix}$$

透视除法后：$(x_{ndc}, y_{ndc}, z_{ndc}) = (x_{clip}/w_{clip}, y_{clip}/w_{clip}, z_{clip}/w_{clip})$

**Unity/DirectX 风格**（NDC $z \in [0, 1]$，左手坐标系）：

$$P_{DX} = \begin{pmatrix} \frac{1}{a \cdot \tan(\theta/2)} & 0 & 0 & 0 \\ 0 & \frac{1}{\tan(\theta/2)} & 0 & 0 \\ 0 & 0 & \frac{f}{f-n} & -\frac{fn}{f-n} \\ 0 & 0 & 1 & 0 \end{pmatrix}$$

### 6.5 反向 Z（Reversed-Z）

**问题**：标准深度缓冲中，近处精度高、远处精度低（非线性分布）。

**解决**：翻转 Z 映射，近平面 → 1，远平面 → 0。

$$z_{reversed} = 1 - z_{standard}$$

修改投影矩阵第三行：

$$P_{reversed} = \begin{pmatrix} \cdots & & & \\ & \cdots & & \\ 0 & 0 & \frac{-n}{f-n} & \frac{fn}{f-n} \\ 0 & 0 & 1 & 0 \end{pmatrix}$$

**精度对比**（32-bit float depth）：

| 距离   | 标准 Z 精度 | 反向 Z 精度 |
| ------ | ----------- | ----------- |
| 1m     | 23 bit      | 23 bit      |
| 10m    | 20 bit      | 23 bit      |
| 100m   | 17 bit      | 23 bit      |
| 1000m  | 14 bit      | 22 bit      |
| 10000m | 10 bit      | 20 bit      |

> **Unity URP/HDRP 默认使用 Reversed-Z**。

### 6.6 深度值线性化

从深度缓冲重建线性深度：

```hlsl
// Standard Z buffer to linear depth
float LinearDepth(float depth, float near, float far) {
    // For standard [0,1] depth (DirectX)
    return near * far / (far - depth * (far - near));
}

// Reversed-Z to linear depth
float LinearDepthReversed(float depth, float near, float far) {
    return near * far / (near + depth * (far - near));
}

// Unity built-in: _ZBufferParams
// x = 1 - far/near
// y = far/near
// z = x/far
// w = y/far
float LinearEyeDepth(float depth) {
    return 1.0 / (_ZBufferParams.z * depth + _ZBufferParams.w);
}
```

## 7. 光照数学模型

### 7.1 Lambert 漫反射

$$L_d = \frac{\rho}{\pi} \cdot L_i \cdot \max(0, \hat{n} \cdot \hat{l})$$

其中 $\rho$ 是 albedo（反射率），$\frac{1}{\pi}$ 是能量守恒归一化因子。

**推导**：Lambert 余弦定律 — 表面接收的辐照度与入射角余弦成正比。

对于完美漫反射表面，BRDF 为常数 $f_d = \frac{\rho}{\pi}$。

验证能量守恒：
$$\int_{\Omega^+} f_d \cos\theta \, d\omega = \frac{\rho}{\pi} \int_0^{2\pi} \int_0^{\pi/2} \cos\theta \sin\theta \, d\theta \, d\phi = \frac{\rho}{\pi} \cdot \pi = \rho \leq 1 \checkmark$$

### 7.2 Blinn-Phong 高光

$$L_s = L_i \cdot (\hat{n} \cdot \hat{h})^{\alpha}$$

其中半角向量 $\hat{h} = \text{normalize}(\hat{l} + \hat{v})$。

**为什么用 Blinn-Phong 而不是 Phong？**

| 对比       | Phong                                                              | Blinn-Phong                   |
| ---------- | ------------------------------------------------------------------ | ----------------------------- |
| 计算       | $\hat{r} = 2(\hat{n} \cdot \hat{l})\hat{n} - \hat{l}$ + `dot(R,V)` | `normalize(L+V)` + `dot(N,H)` |
| 指令数     | 反射向量 ~6 ops                                                    | 半角向量 ~4 ops               |
| 物理正确性 | 较差                                                               | 更接近微表面模型              |
| 高光形状   | 视角掠射时拉伸                                                     | 更自然                        |

### 7.3 Cook-Torrance 微表面 BRDF

$$f_r = \frac{D \cdot F \cdot G}{4(\hat{n} \cdot \hat{l})(\hat{n} \cdot \hat{v})}$$

#### D — 法线分布函数（NDF）

**GGX/Trowbridge-Reitz**（业界标准）：

$$D_{GGX}(\hat{h}) = \frac{\alpha^2}{\pi((\hat{n} \cdot \hat{h})^2(\alpha^2 - 1) + 1)^2}$$

其中 $\alpha = \text{roughness}^2$。

```hlsl
float D_GGX(float NdotH, float roughness) {
    float a = roughness * roughness;
    float a2 = a * a;
    float NdotH2 = NdotH * NdotH;
    float denom = NdotH2 * (a2 - 1.0) + 1.0;
    return a2 / (PI * denom * denom);
}
```

#### F — 菲涅尔项

**Schlick 近似**（替代完整 Fresnel 方程）：

$$F_{Schlick} = F_0 + (1 - F_0)(1 - \hat{v} \cdot \hat{h})^5$$

**完整 Fresnel 方程**（参考）：

$$F_s = \left(\frac{n_1 \cos\theta_i - n_2 \cos\theta_t}{n_1 \cos\theta_i + n_2 \cos\theta_t}\right)^2$$
$$F_p = \left(\frac{n_2 \cos\theta_i - n_1 \cos\theta_t}{n_2 \cos\theta_i + n_1 \cos\theta_t}\right)^2$$
$$F = \frac{F_s + F_p}{2}$$

> Schlick 近似误差 < 1%，节省大量三角函数计算。

```hlsl
float3 F_Schlick(float VdotH, float3 F0) {
    return F0 + (1.0 - F0) * pow(1.0 - VdotH, 5.0);
}

// Optimized: avoid pow, use multiply chain
float3 F_SchlickFast(float VdotH, float3 F0) {
    float x = 1.0 - VdotH;
    float x2 = x * x;
    float x5 = x2 * x2 * x; // 3 multiplications instead of pow
    return F0 + (1.0 - F0) * x5;
}
```

#### G — 几何遮蔽项

**Smith-GGX**：

$$G_{Smith} = G_1(\hat{l}) \cdot G_1(\hat{v})$$

$$G_1(\hat{v}) = \frac{2(\hat{n} \cdot \hat{v})}{(\hat{n} \cdot \hat{v}) + \sqrt{\alpha^2 + (1 - \alpha^2)(\hat{n} \cdot \hat{v})^2}}$$

**性能优化 — 合并 G 和分母**：

$$V = \frac{G}{4(\hat{n} \cdot \hat{l})(\hat{n} \cdot \hat{v})}$$

称为 Visibility 函数，避免分母中的重复计算：

```hlsl
// Combined visibility term (G / denominator)
float V_SmithGGXCorrelated(float NdotV, float NdotL, float roughness) {
    float a2 = roughness * roughness;
    float GGXV = NdotL * sqrt(NdotV * NdotV * (1.0 - a2) + a2);
    float GGXL = NdotV * sqrt(NdotL * NdotL * (1.0 - a2) + a2);
    return 0.5 / (GGXV + GGXL + 1e-5);
}

// Fast approximation (Hammon 2017)
float V_SmithGGXCorrelatedFast(float NdotV, float NdotL, float roughness) {
    float a = roughness;
    float GGXV = NdotL * (NdotV * (1.0 - a) + a);
    float GGXL = NdotV * (NdotL * (1.0 - a) + a);
    return 0.5 / (GGXV + GGXL + 1e-5);
}
```

### 7.4 半球积分与蒙特卡洛

渲染方程：

$$L_o(\vec{\omega}_o) = \int_{\Omega^+} f_r(\vec{\omega}_i, \vec{\omega}_o) L_i(\vec{\omega}_i) (\hat{n} \cdot \vec{\omega}_i) \, d\vec{\omega}_i$$

**蒙特卡洛估计**：

$$L_o \approx \frac{1}{N} \sum_{k=1}^{N} \frac{f_r(\vec{\omega}_k, \vec{\omega}_o) L_i(\vec{\omega}_k) (\hat{n} \cdot \vec{\omega}_k)}{p(\vec{\omega}_k)}$$

其中 $p(\vec{\omega}_k)$ 是采样概率密度函数。

**重要性采样 GGX**：

按 NDF 分布采样半角向量，PDF 为：

$$p(\hat{h}) = \frac{D(\hat{h})(\hat{n} \cdot \hat{h})}{4(\hat{v} \cdot \hat{h})}$$

```hlsl
// GGX importance sampling
float3 ImportanceSampleGGX(float2 Xi, float3 N, float roughness) {
    float a = roughness * roughness;
    
    // Spherical coordinates from random numbers
    float phi = 2.0 * PI * Xi.x;
    float cosTheta = sqrt((1.0 - Xi.y) / (1.0 + (a * a - 1.0) * Xi.y));
    float sinTheta = sqrt(1.0 - cosTheta * cosTheta);
    
    // Tangent space half vector
    float3 H;
    H.x = cos(phi) * sinTheta;
    H.y = sin(phi) * sinTheta;
    H.z = cosTheta;
    
    // Transform to world space
    float3 up = abs(N.z) < 0.999 ? float3(0, 0, 1) : float3(1, 0, 0);
    float3 tangent = normalize(cross(up, N));
    float3 bitangent = cross(N, tangent);
    
    return tangent * H.x + bitangent * H.y + N * H.z;
}
```

## 9. 随机与噪声

### 9.1 伪随机哈希函数

```hlsl
// Simple hash for shader use
float Hash11(float p) {
    p = frac(p * 0.1031);
    p *= p + 33.33;
    p *= p + p;
    return frac(p);
}

float Hash21(float2 p) {
    float3 p3 = frac(float3(p.xyx) * 0.1031);
    p3 += dot(p3, p3.yzx + 33.33);
    return frac((p3.x + p3.y) * p3.z);
}

// Integer hash (better distribution)
uint PCGHash(uint input) {
    uint state = input * 747796405u + 2891336453u;
    uint word = ((state >> ((state >> 28u) + 4u)) ^ state) * 277803737u;
    return (word >> 22u) ^ word;
}
```

### 9.2 Perlin Noise

**核心思想**：在整数格点上定义随机梯度向量，通过插值生成连续噪声。

步骤：
1. 确定所在网格单元
2. 计算到四个（2D）/八个（3D）角点的偏移向量
3. 每个角点有一个伪随机梯度向量
4. 计算偏移向量与梯度向量的点积
5. 用 smootherstep 插值

```hlsl
float PerlinNoise2D(float2 p) {
    float2 i = floor(p);
    float2 f = frac(p);
    
    // Smootherstep interpolation weights
    float2 u = f * f * f * (f * (f * 6.0 - 15.0) + 10.0);
    
    // Random gradients at 4 corners
    float a = dot(RandomGradient(i + float2(0, 0)), f - float2(0, 0));
    float b = dot(RandomGradient(i + float2(1, 0)), f - float2(1, 0));
    float c = dot(RandomGradient(i + float2(0, 1)), f - float2(0, 1));
    float d = dot(RandomGradient(i + float2(1, 1)), f - float2(1, 1));
    
    return lerp(lerp(a, b, u.x), lerp(c, d, u.x), u.y);
}
```

### 9.3 FBM（分形布朗运动）

$$\text{FBM}(p) = \sum_{i=0}^{N-1} A^i \cdot \text{noise}(F^i \cdot p)$$

其中 $A$ 是振幅衰减（通常 0.5），$F$ 是频率倍增（通常 2.0）。

```hlsl
float FBM(float2 p, int octaves) {
    float value = 0.0;
    float amplitude = 0.5;
    float frequency = 1.0;
    
    for (int i = 0; i < octaves; i++) {
        value += amplitude * PerlinNoise2D(p * frequency);
        amplitude *= 0.5;  // persistence
        frequency *= 2.0;  // lacunarity
    }
    return value;
}
```

**性能优化**：
- 低端机减少 octaves（8 → 3）
- 远距离物体使用更少 octaves
- 预烘焙到纹理（离线计算）

### 9.4 Voronoi / Worley Noise

```hlsl
float VoronoiNoise(float2 p) {
    float2 i = floor(p);
    float2 f = frac(p);
    
    float minDist = 1e10;
    
    // Check 3x3 neighborhood
    for (int y = -1; y <= 1; y++) {
        for (int x = -1; x <= 1; x++) {
            float2 neighbor = float2(x, y);
            float2 point = Hash22(i + neighbor); // random point in cell
            float2 diff = neighbor + point - f;
            float dist = dot(diff, diff); // squared distance
            minDist = min(minDist, dist);
        }
    }
    return sqrt(minDist);
}
```

### 9.5 低差异序列

**Halton 序列**（用于蒙特卡洛采样）：

```hlsl
float RadicalInverse(uint bits) {
    bits = (bits << 16u) | (bits >> 16u);
    bits = ((bits & 0x55555555u) << 1u) | ((bits & 0xAAAAAAAAu) >> 1u);
    bits = ((bits & 0x33333333u) << 2u) | ((bits & 0xCCCCCCCCu) >> 2u);
    bits = ((bits & 0x0F0F0F0Fu) << 4u) | ((bits & 0xF0F0F0F0u) >> 4u);
    bits = ((bits & 0x00FF00FFu) << 8u) | ((bits & 0xFF00FF00u) >> 8u);
    return float(bits) * 2.3283064365386963e-10; // / 0x100000000
}

float2 Hammersley(uint i, uint N) {
    return float2(float(i) / float(N), RadicalInverse(i));
}
```

| `atan2(y,x)`   | 很高           | 多项式近似          |
| `sin/cos`      | 中             | 查找表 / 多项式近似 |
| `normalize(v)` | 中（含 rsqrt） | 已知单位向量时跳过  |
| `1.0 / x`      | 中             | `rcp(x)` 硬件指令   |
| `exp(x)`       | 中             | `exp2(x * 1.4427)`  |
| `log(x)`       | 中             | `log2(x) * 0.6931`  |

### 10.2 常用数学替代

```hlsl
// 1. Replace pow with multiply chain
// BAD
float r = pow(x, 5.0);
// GOOD
float x2 = x * x;
float r = x2 * x2 * x; // 3 mul vs pow (~10 cycles)

// 2. Replace exp with exp2
// exp(x) = exp2(x * LOG2E)
float r = exp2(x * 1.44269504089); // exp2 is native on GPU

// 3. Replace log with log2
// log(x) = log2(x) * LN2
float r = log2(x) * 0.69314718056;

// 4. Replace division with multiplication
// BAD
float r = a / b;
// GOOD (when b is reused)
float invB = rcp(b); // or 1.0 / b
float r1 = a * invB;
float r2 = c * invB;

// 5. Saturate is free on most GPUs
float r = max(0.0, min(1.0, x)); // costs instructions
float r = saturate(x);            // free modifier

// 6. mad (multiply-add) is single instruction
float r = a * b + c; // Compiler usually optimizes, but be explicit

// 7. Avoid branching in pixel shaders
// BAD
if (x > 0) r = a;
else r = b;
// GOOD
r = lerp(b, a, step(0, x));
// BETTER (if both paths are cheap)
r = x > 0 ? a : b; // Ternary often compiles to conditional move

// 8. Pack/unpack normals
// Encode normal to [0,1] for storage
float2 EncodeNormalOctahedron(float3 n) {
    n /= (abs(n.x) + abs(n.y) + abs(n.z));
    if (n.z < 0) {
        n.xy = (1.0 - abs(n.yx)) * sign(n.xy);
    }
    return n.xy * 0.5 + 0.5;
}

float3 DecodeNormalOctahedron(float2 f) {
    f = f * 2.0 - 1.0;
    float3 n = float3(f.xy, 1.0 - abs(f.x) - abs(f.y));
    if (n.z < 0) {
        n.xy = (1.0 - abs(n.yx)) * sign(n.xy);
    }
    return normalize(n);
}
```

### 10.4 矩阵运算优化

```hlsl
// 1. 3x4 matrix instead of 4x4 (save 25% memory & bandwidth)
// When last row is always (0,0,0,1)
struct float3x4_packed {
    float4 row0; // right + tx
    float4 row1; // up + ty  
    float4 row2; // forward + tz
};

float3 TransformPoint(float3x4_packed m, float3 p) {
    return float3(
        dot(m.row0, float4(p, 1)),
        dot(m.row1, float4(p, 1)),
        dot(m.row2, float4(p, 1))
    );
}

// 2. Transform direction (skip translation)
float3 TransformDirection(float3x4_packed m, float3 d) {
    return float3(
        dot(m.row0.xyz, d),
        dot(m.row1.xyz, d),
        dot(m.row2.xyz, d)
    );
}
```

### 10.5 视锥剔除数学

**平面方程**：$ax + by + cz + d = 0$，其中 $(a,b,c)$ 是法线。

**点到平面距离**：
$$\text{dist} = \hat{n} \cdot \vec{p} + d$$

**球体-视锥测试**：

```hlsl
// Test sphere against 6 frustum planes
bool SphereFrustumTest(float3 center, float radius, float4 planes[6]) {
    for (int i = 0; i < 6; i++) {
        float dist = dot(planes[i].xyz, center) + planes[i].w;
        if (dist < -radius)
            return false; // completely outside
    }
    return true; // inside or intersecting
}

// Extract frustum planes from VP matrix
void ExtractFrustumPlanes(float4x4 VP, out float4 planes[6]) {
    // Left:   row3 + row0
    planes[0] = VP[3] + VP[0];
    // Right:  row3 - row0
    planes[1] = VP[3] - VP[0];
    // Bottom: row3 + row1
    planes[2] = VP[3] + VP[1];
    // Top:    row3 - row1
    planes[3] = VP[3] - VP[1];
    // Near:   row3 + row2
    planes[4] = VP[3] + VP[2];
    // Far:    row3 - row2
    planes[5] = VP[3] - VP[2];
    
    // Normalize planes
    for (int i = 0; i < 6; i++) {
        float len = length(planes[i].xyz);
        planes[i] /= len;
    }
}
```

### 10.6 AABB-平面测试优化

```hlsl
// Optimized AABB vs plane test
// Only test the vertex most aligned with plane normal (p-vertex)
bool AABBPlaneTest(float3 bmin, float3 bmax, float4 plane) {
    // Select p-vertex (furthest along normal)
    float3 pVertex = float3(
        plane.x > 0 ? bmax.x : bmin.x,
        plane.y > 0 ? bmax.y : bmin.y,
        plane.z > 0 ? bmax.z : bmin.z
    );
    return dot(plane.xyz, pVertex) + plane.w >= 0;
}

// Full AABB-Frustum test
bool AABBFrustumTest(float3 bmin, float3 bmax, float4 planes[6]) {
    for (int i = 0; i < 6; i++) {
        if (!AABBPlaneTest(bmin, bmax, planes[i]))
            return false;
    }
    return true;
}
```

### 光照计算最小集

```hlsl
// Minimal PBR lighting
float3 PBR_DirectLight(float3 N, float3 V, float3 L, float3 albedo, 
                       float metallic, float roughness, float3 lightColor) {
    float3 H = normalize(L + V);
    float NdotL = max(0, dot(N, L));
    float NdotV = max(1e-5, dot(N, V));
    float NdotH = max(0, dot(N, H));
    float VdotH = max(0, dot(V, H));
    
    // F0 for dielectrics ~0.04, for metals use albedo
    float3 F0 = lerp(0.04, albedo, metallic);
    
    // Specular BRDF
    float D = D_GGX(NdotH, roughness);
    float3 F = F_SchlickFast(VdotH, F0);
    float Vis = V_SmithGGXCorrelatedFast(NdotV, NdotL, roughness);
    float3 specular = D * F * Vis;
    
    // Diffuse BRDF (energy conserving)
    float3 kD = (1.0 - F) * (1.0 - metallic);
    float3 diffuse = kD * albedo * INV_PI;
    
    return (diffuse + specular) * lightColor * NdotL;
}
```
