# 1. 基础知识

## 1.1 卡通 PBR 概念

卡通 PBR（Toon PBR / Stylized PBR）是将物理渲染（PBR）与卡通风格化（Toon Shading）结合的渲染方案：

- **PBR 部分**：使用物理正确的光照模型（GGX、Cook-Torrance），保证光照在不同环境下的一致性
- **卡通部分**：对漫反射进行阶梯化（Step/Ramp），对高光进行锐化，增加描边、边缘光等风格化处理

```
真实 PBR ←──────────────────────────────→ 纯卡通
          卡通PBR（Stylized PBR）在中间地带
          - 物理高光（GGX）
          - 阶梯漫反射（Ramp）
          - 风格化描边
          - 边缘光（Rim）
```

## 1.2 核心光照模型

### Lambert 漫反射（卡通化）

```hlsl
// 标准 Lambert
float NdotL = saturate(dot(N, L));

// 卡通化：阶梯化漫反射（Ramp）
float toonDiff = smoothstep(0.4, 0.5, NdotL);  // 硬边
// 或使用 Ramp 贴图
float toonDiff = tex2D(_RampTex, float2(NdotL, 0)).r;
```

### GGX 高光（物理）

```hlsl
// Cook-Torrance BRDF
float3 H = normalize(L + V);
float NdotH = saturate(dot(N, H));
float NdotL = saturate(dot(N, L));
float NdotV = saturate(dot(N, V));

// GGX 法线分布函数
float D = D_GGX(NdotH, roughness);
// Schlick 菲涅尔
float3 F = F_Schlick(specColor, dot(H, V));
// Smith 几何遮蔽
float G = G_Smith(NdotV, NdotL, roughness);

float3 specular = D * F * G / (4.0 * NdotL * NdotV + 1e-5);
```

### 卡通光照公式

$$
L_{\text{total}} = I_{\text{ambient}} \times K_{\text{ambient}} + I_{\text{diffuse}} \times K_{\text{diffuse}} \times \text{step}(0.5, \mathbf{N} \cdot \mathbf{L}) + I_{\text{specular}} \times K_{\text{specular}} \times \text{step}(0.95, \mathbf{R} \cdot \mathbf{V})
$$

## 1.3 贴图规范

| 贴图类型       | 通道                               | 说明                           |
| -------------- | ---------------------------------- | ------------------------------ |
| Albedo         | RGB                                | 基础色，不含光照信息           |
| Normal         | RGB                                | 切线空间法线贴图               |
| Mask           | R=金属度, G=粗糙度, B=AO, A=自发光 | 打包贴图，减少采样次数         |
| SSS/Thickness  | R                                  | 次表面散射厚度，皮肤/耳朵/鼻尖 |
| Hair Direction | RG                                 | 头发流向，各向异性高光方向     |
| Ramp           | RGB                                | 漫反射渐变控制贴图             |

## 1.4 渲染顺序

```
Opaque（不透明）
  ├── 皮肤（Skin）
  ├── 眼睛内层（Iris/Sclera）
  ├── 头发 Pass1（Depth Prepass + Alpha Test）
  └── 毛发（Fur Shell 1~N）

Transparent（透明）
  ├── 头发 Pass2（Alpha Blend 柔和边缘）
  ├── 眼睛外层（Cornea，半透明）
  └── 泪膜（Tear Line）

Post Process（后处理）
  └── 屏幕空间描边（可选）
```

# 2. 头发渲染

## 2.1 物理特征

- **各向异性反射**：头发表面鳞片结构使高光沿切线方向拉伸，而非圆形高光
- **多层结构**：底层（主发）、主层（发束）、飞丝层（边缘细发）
- **半透明**：发束边缘半透明，需要 Alpha Blending
- **自遮挡**：发束之间相互遮挡，需要正确的深度排序

## 2.2 各向异性高光原理

```
各向异性反射（Anisotropic Reflection）：
  物体表面对光的反射在不同方向上表现不同
  头发高光不是圆点，而是沿发丝方向拉伸成线

参数说明：
  各向异性值越高（偏离0），高光越拉长（丝绸/拉丝金属效果）
  低值（0）= 各向同性，传统圆形高光
```

**Kajiya-Kay 各向异性模型（GDC 2004）：**

```hlsl
// T: 头发切线方向（沿发丝方向）
// N: 法线, L: 光向, V: 视线, H: 半角向量

// 切线偏移（用法线偏移切线，模拟鳞片起伏）
half3 ShiftTangent(half3 T, half3 N, float shift) {
    half3 shiftedT = T + shift * N;
    return normalize(shiftedT);
}

// 各向异性高光核心
float StrandSpecular(half3 T, half3 V, half3 L, float exponent) {
    half3 H = normalize(L + V);
    float dotTH = dot(T, H);
    float sinTH = sqrt(1.0 - dotTH * dotTH);  // sin²+cos²=1
    float dirAtten = smoothstep(-1, 0, dotTH); // 方向衰减
    return dirAtten * pow(sinTH, exponent);
}

// 完整头发光照
float4 HairLighting(float3 tangent, float3 normal, float3 lightVec,
                    float3 viewVec, float2 uv, float ambOcc) {
    // 从贴图读取切线偏移（模拟鳞片随机性）
    float shiftTex = tex2D(_SpecShiftTex, uv).r - 0.5;

    // 主高光切线（偏移1）
    float3 t1 = ShiftTangent(tangent, normal, _PrimaryShift + shiftTex);
    // 次高光切线（偏移2，模拟内反射）
    float3 t2 = ShiftTangent(tangent, normal, _SecondaryShift + shiftTex);

    // 漫反射（各向异性调整，wrap lighting）
    float3 diffuse = saturate(lerp(0.25, 1.0, dot(normal, lightVec)));

    // 主高光
    float3 specular = _SpecColor1 * StrandSpecular(t1, viewVec, lightVec, _SpecExp1);

    // 次高光（带遮罩）
    float specMask = tex2D(_SpecMaskTex, uv).r;
    specular += _SpecColor2 * specMask * StrandSpecular(t2, viewVec, lightVec, _SpecExp2);

    float4 o;
    o.rgb = (diffuse + specular) * tex2D(_BaseTex, uv).rgb * _LightColor;
    o.rgb *= ambOcc;
    o.a = tex2D(_AlphaTex, uv).r;
    return o;
}
```

## 2.3 双 Pass 渲染流程

头发同时需要硬边（Alpha Test）和柔和边缘（Alpha Blend），使用双 Pass 解决：

```
Pass 1 - Depth Prepass（深度预写入）
  ├── Cull Back
  ├── ZWrite On
  ├── ColorMask 0（不输出颜色）
  └── Alpha Test（AlphaClip）→ 写入正确深度

Pass 2 - Color Pass（颜色输出）
  ├── Cull Off（双面渲染）
  ├── ZWrite Off
  ├── ZTest LEqual（利用 Pass1 写入的深度）
  ├── Blend SrcAlpha OneMinusSrcAlpha
  └── 完整光照计算 + Alpha Blend
```

**优化原理：**
- Pass1 用 Alpha Test 写入深度 → Early Z 剔除大量被遮挡片元
- Pass2 用 Alpha Blend 渲染柔和边缘 → 避免硬边锯齿
- 先画头（Head Mesh）→ 用 Early Z 剔除头发穿透头部的部分

```hlsl
// Pass 1: Depth Prepass
Pass {
    Name "DepthPrepass"
    ColorMask 0
    ZWrite On
    Cull Back
    HLSLPROGRAM
    // 只做 Alpha Test，不输出颜色
    clip(tex2D(_AlphaTex, uv).r - _Cutoff);
    ENDHLSL
}

// Pass 2: Hair Shading
Pass {
    Name "HairForward"
    ZWrite Off
    ZTest LEqual
    Cull Off
    Blend SrcAlpha OneMinusSrcAlpha
    HLSLPROGRAM
    // 完整头发光照
    ENDHLSL
}
```

## 2.4 头发流向贴图（Hair Direction Map）

**生成流程：**

```
1. DCC 软件（Maya/Blender）制作头发卡片（Hair Cards）
   └── 按自然流向梳理，UV 的 U 方向 = 发根到发梢

2. UV 展开
   └── 确保 UV 流向与头发生长方向一致

3. 流向贴图生成（二选一）：
   ├── 手绘：Photoshop/Substance Painter 手绘流向线条
   │         R=U方向分量, G=V方向分量, B=备用
   └── 自动：Blender Hair Tool / Maya XGen 自动烘焙切线方向

4. 导出 PNG/TGA（8位或16位）
   └── Shader 中采样，作为各向异性高光的切线方向输入
```

**颜色编码：**

| 通道 | 含义                  |
| ---- | --------------------- |
| R    | 切线 X 分量（U 方向） |
| G    | 切线 Y 分量（V 方向） |
| B    | 分组 ID 或备用        |

## 2.5 优化方案

| 优化项        | 方案                         | 说明                                 |
| ------------- | ---------------------------- | ------------------------------------ |
| Early Z       | 双 Pass + Depth Prepass      | 先写深度，后渲染颜色，节省 Fill Rate |
| 先画头        | 头部 Mesh 先渲染             | Early Z 剔除头发穿透头部的片元       |
| 减少 Overdraw | 合理分层，减少飞丝层数       | 飞丝层控制在 2~3 层                  |
| LOD           | 远距离减少 Hair Card 数量    | 3~4 级 LOD                           |
| 贴图压缩      | Alpha 贴图使用 BC4（单通道） | 减少带宽                             |
| 合批          | 同材质头发合并 DrawCall      | GPU Instancing                       |

# 3. 皮肤渲染

## 3.1 物理结构

```
皮肤多层结构：
┌─────────────────────────────┐
│  油脂层（Oily Layer）        │ ← 高光反射（Specular）
├─────────────────────────────┤
│  表皮层（Epidermis）         │ ← 漫反射 + 浅层 SSS
├─────────────────────────────┤
│  真皮层（Dermis）            │ ← 深层红色 SSS（血色）
├─────────────────────────────┤
│  皮下组织（Hypodermis）      │ ← 深层透光（耳朵/鼻尖）
└─────────────────────────────┘
```

## 3.2 次表面散射（SSS）

光线穿透表皮后在内部多次散射，形成柔和的血色和半透明效果（如手指透光、耳朵透光）。

**实现方式对比：**

| 方式            | 质量  | 性能  | 适用场景         |
| --------------- | ----- | ----- | ---------------- |
| 高斯模糊近似    | ★★☆   | ★★★★★ | 移动端、低端机   |
| 厚度贴图 + 透射 | ★★★   | ★★★★☆ | 中端机、卡通风格 |
| 屏幕空间 SSS    | ★★★★  | ★★★☆☆ | PC/主机          |
| BSSRDF（离线）  | ★★★★★ | ★☆☆☆☆ | 离线渲染         |

**厚度贴图 SSS（推荐，移动端友好）：**

```hlsl
// 厚度贴图：耳朵/鼻尖/手指 = 高值（更透光）
float thickness = tex2D(_ThicknessMap, uv).r;

// 透射光（背光穿透）
float3 transLightDir = L + N * _TransDistortion;
float transDot = pow(saturate(dot(V, -transLightDir)), _TransPower) * _TransScale;
float3 transmission = transDot * thickness * _SSSColor * lightColor;

// 前向散射（正面柔和）
float3 sss = GaussianBlur(_MainTex, uv, _SSSBlurSize) * _SSSColor * thickness;
```

## 3.3 皮肤 BRDF 完整流程

```hlsl
// 输入贴图
float3 albedo    = tex2D(_MainTex, uv).rgb;
float3 normal    = SampleNormal(uv);
float  thickness = tex2D(_ThicknessMap, uv).r;   // SSS 厚度
float  oiliness  = tex2D(_OilyMask, uv).r;       // 油脂遮罩（高光区域）
float  roughness = tex2D(_MaskTex, uv).g;

float3 L = normalize(lightDir);
float3 V = normalize(viewDir);
float3 H = normalize(L + V);
float NdotL = saturate(dot(normal, L));
float NdotV = saturate(dot(normal, V));
float NdotH = saturate(dot(normal, H));

// 1. 漫反射（Lambert）
float3 diffuse = albedo * NdotL;

// 2. SSS（高斯模糊 Albedo 近似）
float3 blurredAlbedo = GaussianBlur(_MainTex, uv, _SSSBlurSize);
float3 sss = blurredAlbedo * _SSSColor * _SSSStrength;

// 3. 高光（GGX Cook-Torrance）
float D = D_GGX(NdotH, roughness);
float3 F = F_Schlick(_SpecColor, dot(H, V));
float G = G_Smith(NdotV, NdotL, roughness);
float3 specular = D * F * G / (4.0 * NdotL * NdotV + 1e-5);
specular *= oiliness;  // 油脂区域才有高光

// 4. 菲涅尔边缘光（Rim）
float rim = pow(1.0 - NdotV, _RimPower) * _RimIntensity;

// 5. 组合
// finalColor = lerp(diffuse, sss, SSSMask) + specular + rim * RimColor
float3 finalColor = lerp(diffuse, sss, thickness * _SSSMask)
                  + specular
                  + rim * _RimColor;
```

**分层 BRDF 公式：**

```
finalColor =
    Diffuse_Lambert  × (1 - SSS_Mask)   // 普通漫反射区域
  + SSS_BRDF         × SSS_Mask         // SSS 贡献区域
  + Specular_GGX     × Oil_Mask         // 油脂高光区域
  + Rim              × Fresnel          // 菲涅尔边缘光
```

## 3.4 贴图制作要点

| 贴图          | 说明                        | 制作工具          |
| ------------- | --------------------------- | ----------------- |
| Albedo        | 皮肤基础色，含毛孔/雀斑细节 | Substance Painter |
| Normal        | 法线贴图，含毛孔/皱纹细节   | ZBrush 烘焙       |
| Thickness     | 厚度图，耳朵/鼻尖/手指高值  | Substance Painter |
| Oily Mask     | 油脂遮罩，T区/额头高值      | 手绘              |
| Detail Normal | 微观法线（毛孔级别）        | Substance Painter |

## 3.5 优化方案

| 优化项   | 方案                          | 说明                         |
| -------- | ----------------------------- | ---------------------------- |
| SSS 近似 | 厚度贴图代替屏幕空间模糊      | 移动端性能友好               |
| 贴图合并 | Mask 贴图打包 4 通道          | 减少采样次数                 |
| 法线模糊 | 预模糊法线贴图                | 避免"塑料感"，减少运行时计算 |
| LOD      | 远距离简化 SSS 计算           | 关闭 SSS，使用简单漫反射     |
| 高光简化 | 移动端用 Blinn-Phong 代替 GGX | 减少 ALU 指令                |

# 4. 眼睛渲染

## 4.1 眼球结构

```
眼球分层模型：
┌──────────────────────────────────────┐
│  角膜（Cornea）                       │ ← 外层透明，高折射率（IOR≈1.38）
│  高光 + 环境反射 + 折射               │
├──────────────────────────────────────┤
│  虹膜（Iris）                         │ ← 有纹理和色彩，带法线细节
│  漫反射 + 微弱高光                    │
├──────────────────────────────────────┤
│  瞳孔（Pupil）                        │ ← Alpha Mask 控制，完全黑色
├──────────────────────────────────────┤
│  巩膜（Sclera）                       │ ← 眼白，带血管纹理，湿润感
└──────────────────────────────────────┘
泪膜（Tear Line）：眼睑边缘高光带，增强湿润感
```

## 4.2 渲染流程

**模型分层：**
- **外层 Mesh**：角膜（透明壳，略大于眼球）
- **内层 Mesh**：虹膜 + 巩膜（带贴图）
- **泪膜 Mesh**：眼睑边缘（可选，半透明）

**Shader 流程：**

```hlsl
// ===== 外层角膜 =====
// 1. 菲涅尔反射（边缘更亮）
float fresnel = pow(1.0 - dot(N, V), _FresnelPower);
// 2. 环境反射（Cubemap）
float3 envReflect = texCUBE(_EnvCubemap, reflect(-V, N)).rgb;
// 3. 折射（Parallax 近似，虹膜略微变形）
float2 refractOffset = N.xy * _RefractionStrength;
float2 irisUV = uv + refractOffset;
// 4. 角膜高光（GGX，低粗糙度）
float3 corneaSpec = SpecularGGX(N, L, V, _CorneaRoughness) * _CorneaSpecColor;
// 角膜颜色
float3 corneaColor = lerp(envReflect, corneaSpec, fresnel);

// ===== 内层虹膜/巩膜 =====
float3 irisAlbedo = tex2D(_IrisTex, irisUV).rgb;
float3 irisNormal = SampleNormal(_IrisNormalTex, irisUV);
float3 irisDiffuse = irisAlbedo * saturate(dot(irisNormal, L));
// 厚度遮罩（虹膜外围略暗）
float irisMask = tex2D(_IrisMaskTex, uv).r;
float3 irisColor = irisDiffuse * irisMask;

// ===== 瞳孔 =====
float pupilMask = tex2D(_PupilMaskTex, uv).r;
irisColor *= (1.0 - pupilMask);  // 瞳孔区域变黑

// ===== 最终合成 =====
float corneaAlpha = fresnel * _CorneaOpacity;
float3 finalColor = lerp(irisColor, corneaColor, corneaAlpha);
```

## 4.3 折射近似（Parallax Mapping）

```hlsl
// 用视差贴图近似角膜折射效果
// 视线方向在切线空间的偏移 → 虹膜 UV 偏移
float3 viewTS = mul(TBN_Inv, V);  // 视线转到切线空间
float2 parallaxOffset = viewTS.xy / viewTS.z * _ParallaxDepth;
float2 irisUV = uv - parallaxOffset;  // 偏移后的虹膜 UV
```

## 4.4 动态控制

```hlsl
// 瞳孔大小（根据光照强度动态变化）
float lightIntensity = dot(lightColor, float3(0.299, 0.587, 0.114));
float pupilSize = lerp(_PupilMinSize, _PupilMaxSize, 1.0 - lightIntensity);
// 用 pupilSize 控制瞳孔遮罩的缩放

// 眼球转动（控制虹膜 UV 偏移）
float2 irisOffset = _EyeDirection.xy * _IrisOffsetStrength;
```

## 4.5 优化方案

| 优化项   | 方案                      | 说明                 |
| -------- | ------------------------- | -------------------- |
| 折射简化 | Parallax 代替真实折射     | 移动端性能友好       |
| 环境反射 | 低分辨率 Cubemap（64x64） | 眼睛反射不需要高精度 |
| 泪膜     | 仅主角/近景开启           | 远景关闭泪膜 Mesh    |
| 贴图合并 | 虹膜/巩膜合并一张贴图     | 减少 DrawCall        |
| LOD      | 远距离关闭折射和环境反射  | 简化为基础漫反射     |

# 5. 描边渲染

## 5.1 描边种类总览

| 描边种类               | 主要作用   | 实现方式           | 性能开销        |
| ---------------------- | ---------- | ------------------ | --------------- |
| 轮廓扩展法（顶点描边） | 外轮廓分界 | 法线反向扩展       | 低（额外 Pass） |
| 屏幕空间描边           | 全场景边缘 | Sobel/深度法线检测 | 中（后处理）    |
| 内部边缘描边           | 内部结构   | 法线夹角检测       | 中（后处理）    |
| 纹理描边               | 贴图线稿   | 手绘贴图叠加       | 极低            |
| 色块边缘描边           | 区域分界   | 材质 ID 检测       | 中              |

## 5.2 顶点描边（轮廓扩展法）

最常用的卡通描边方案，额外渲染一个背面扩展的 Pass：

```hlsl
// 描边 Pass（Cull Front，只渲染背面）
Pass {
    Name "Outline"
    Cull Front
    ZWrite On
    ZTest LEqual

    HLSLPROGRAM
    float _OutlineWidth;
    float4 _OutlineColor;

    Varyings vert(Attributes v) {
        Varyings o;
        // 在裁剪空间沿法线外扩（屏幕空间等宽描边）
        float4 clipPos = TransformObjectToHClip(v.positionOS);
        float3 clipNormal = mul((float3x3)UNITY_MATRIX_VP,
                                mul((float3x3)UNITY_MATRIX_M, v.normalOS));
        // 归一化到屏幕空间，保持描边宽度一致
        float2 offset = normalize(clipNormal.xy) * _OutlineWidth
                      / _ScreenParams.xy * clipPos.w * 2.0;
        clipPos.xy += offset;
        o.positionCS = clipPos;
        return o;
    }

    float4 frag(Varyings i) : SV_Target {
        return _OutlineColor;
    }
    ENDHLSL
}
```

**屏幕空间等宽描边（推荐）：**

```hlsl
// 在裁剪空间做偏移，保证远近描边宽度一致
float2 offset = normalize(clipNormal.xy) * _OutlineWidth / _ScreenParams.xy * clipPos.w * 2.0;
clipPos.xy += offset;
```

## 5.3 边缘光（Rim Light）

```hlsl
// 通过法向量和视向量夹角判定 Rim 强度
// 乘上衰减后只对直接光照区表现 Rim
half3 ToonRim(half3 NormalDir, half3 ViewDir, half3 LightDir,
              half3 LightColor, float shadowAtten) {
    float rimArea = 1.0 - saturate(dot(NormalDir, ViewDir));
    // 只在受光面显示 Rim（避免背光面出现 Rim）
    rimArea *= (0.5 * dot(NormalDir, LightDir) + 0.5) * saturate(shadowAtten);
    return _RimColor * LightColor
         * smoothstep(1 - _RimOffset, 1 - _RimOffset + _RimFeatherLevel * _RimOffset, rimArea);
}
```

## 5.4 屏幕空间描边（后处理）

适合全场景统一描边，不侵入模型几何：

```hlsl
// 后处理 Shader：基于深度 + 法线的边缘检测
fixed4 frag(v2f_img i) : SV_Target {
    float2 uv = i.uv;
    float3 centerNormal = UnpackNormal(tex2D(_CameraDepthNormalsTexture, uv));
    float centerDepth = Linear01Depth(tex2D(_CameraDepthNormalsTexture, uv).a);

    float edge = 0.0;
    float2 offset = float2(_Thickness / _ScreenParams.x, _Thickness / _ScreenParams.y);

    // 3x3 邻域采样，检测法线/深度突变
    for (int y = -1; y <= 1; ++y) {
        for (int x = -1; x <= 1; ++x) {
            if (x == 0 && y == 0) continue;
            float2 sampleUV = uv + float2(x, y) * offset;
            float3 sampleNormal = UnpackNormal(tex2D(_CameraDepthNormalsTexture, sampleUV));
            float sampleDepth = Linear01Depth(tex2D(_CameraDepthNormalsTexture, sampleUV).a);

            edge = max(edge, 1.0 - saturate(dot(centerNormal, sampleNormal)));
            edge = max(edge, abs(centerDepth - sampleDepth));
        }
    }

    if (edge > _Threshold)
        return _OutlineColor;
    else
        return tex2D(_MainTex, uv);
}
```

## 5.5 描边优化方案

| 优化项   | 方案                             | 说明                 |
| -------- | -------------------------------- | -------------------- |
| 描边宽度 | 屏幕空间等宽（clipPos.w 补偿）   | 远近描边宽度一致     |
| 法线平滑 | 导入时平滑法线，存入 UV2/切线    | 避免硬边模型描边断裂 |
| 移动端   | 优先顶点描边，避免屏幕空间后处理 | 减少带宽和 Fill Rate |
| LOD      | 远距离关闭描边 Pass              | 节省 DrawCall        |
| 描边颜色 | 根据材质/区域变化（非纯黑）      | 提升卡通质感         |

# 6. 毛发渲染

## 6.1 Shell-Based Fur（壳层法）

最常用的实时毛发方案，通过多层偏移 Mesh 模拟毛发体积：

```
Shell 层示意：
  Layer 0（最内层，紧贴皮肤）
  Layer 1
  Layer 2
  ...
  Layer N（最外层，毛发尖端）

每层沿法线方向偏移 = 毛发长度 / N
每层 Alpha 随层数增加而减小（尖端透明）
```

```hlsl
// Shell Fur Shader 核心
Properties {
    _FurLength ("Fur Length", Range(0, 0.5)) = 0.1
    _FurDensity ("Fur Density", Range(0, 100)) = 50
    _ShellIndex ("Shell Index", Range(0, 1)) = 0  // 当前层（0=根部, 1=尖端）
    _FurTex ("Fur Noise Texture", 2D) = "white" {}
}

// 顶点着色器：沿法线偏移
Varyings vert(Attributes v) {
    float3 worldPos = TransformObjectToWorld(v.positionOS);
    float3 worldNormal = TransformObjectToWorldNormal(v.normalOS);
    // 沿法线偏移（加重力影响）
    float3 gravity = float3(0, -1, 0) * _GravityStrength * _ShellIndex;
    worldPos += (worldNormal + gravity) * _FurLength * _ShellIndex;
    o.positionCS = TransformWorldToHClip(worldPos);
    return o;
}

// 片元着色器：噪声裁剪
float4 frag(Varyings i) : SV_Target {
    float furNoise = tex2D(_FurTex, i.uv * _FurDensity).r;
    // 尖端裁剪（层数越高，裁剪阈值越高）
    clip(furNoise - _ShellIndex);
    // 颜色随层数变化（根部深，尖端浅）
    float3 color = lerp(_RootColor, _TipColor, _ShellIndex);
    // AO（根部更暗）
    float ao = _ShellIndex;
    return float4(color * ao, 1.0);
}
```

## 6.2 Fin-Based Fur（鳍片法）

在 Shell 基础上，在轮廓处添加垂直于表面的"鳍片"，增强侧面毛发效果：

```
Fin 示意（侧视图）：
  ┌─────────────────┐  ← 模型表面
  │  Shell Layers   │
  │  ↑↑↑↑↑↑↑↑↑↑↑    │  ← 法线方向毛发
  └─────────────────┘
  ↕↕↕↕↕↕↕↕↕↕↕↕↕↕↕  ← Fin（轮廓处垂直片）
```

## 6.3 优化方案

| 优化项     | 方案                        | 说明                                 |
| ---------- | --------------------------- | ------------------------------------ |
| Shell 层数 | 移动端 8~12 层，PC 16~32 层 | 层数越多越真实，但 DrawCall 成倍增加 |
| 合并 Shell | 使用 Geometry Shader 生成   | 减少 CPU DrawCall（需 GPU 支持）     |
| LOD        | 远距离减少 Shell 层数       | 距离 > 5m 减半层数                   |
| 裁剪优化   | 早期 clip() 丢弃空白片元    | 减少 Fill Rate                       |
| 贴图       | 使用噪声贴图代替逐根计算    | 性能友好                             |

# 7. 整体优化方案

## 7.1 DrawCall 优化

```
角色渲染 DrawCall 分析：
  皮肤：1 DC
  眼睛：2 DC（内层 + 外层角膜）
  头发：2 DC（Depth Prepass + Color Pass）
  描边：1 DC（顶点描边 Pass）
  毛发：N DC（Shell 层数）
  ─────────────────────────────
  总计：6 + N DC（N = Shell 层数）

优化策略：
  1. 合并皮肤/眼白/眼睑为同一材质（减少 DC）
  2. GPU Instancing（多角色同材质合批）
  3. SRP Batcher（Unity URP 自动合批）
  4. 毛发 Shell 使用 Geometry Shader 合并
```

## 7.3 贴图优化

| 贴图       | PC/主机     | 移动端高端       | 移动端低端      |
| ---------- | ----------- | ---------------- | --------------- |
| Albedo     | BC7（2048） | ASTC 6x6（1024） | ASTC 8x8（512） |
| Normal     | BC5（2048） | ASTC 6x6（1024） | ASTC 8x8（512） |
| Mask       | BC7（1024） | ASTC 6x6（512）  | ASTC 8x8（256） |
| Hair Alpha | BC4（1024） | ASTC 6x6（512）  | ASTC 8x8（256） |

## 7.4 LOD 策略

```
LOD 0（< 3m）：完整渲染
  ├── 皮肤：完整 SSS + GGX 高光
  ├── 头发：双 Pass + 各向异性高光
  ├── 眼睛：折射 + 环境反射
  ├── 描边：顶点描边
  └── 毛发：16 Shell 层

LOD 1（3~8m）：中等质量
  ├── 皮肤：厚度贴图 SSS（关闭高斯模糊）
  ├── 头发：单 Pass + 简化高光
  ├── 眼睛：关闭折射
  ├── 描边：保留
  └── 毛发：8 Shell 层

LOD 2（8~15m）：低质量
  ├── 皮肤：Lambert + 简单高光
  ├── 头发：Alpha Test 单 Pass
  ├── 眼睛：贴图直接输出
  ├── 描边：关闭
  └── 毛发：关闭

LOD 3（> 15m）：极简
  └── 合并 Mesh + 单材质
```

## 7.5 渲染管线选择

| 管线                       | 适用场景      | 角色渲染支持               |
| -------------------------- | ------------- | -------------------------- |
| URP（Universal RP）        | 移动端/中低端 | 自定义 Pass、SRP Batcher   |
| HDRP（High Definition RP） | PC/主机/高端  | 内置 SSS、Eye、Hair Shader |
| Built-in                   | 旧项目维护    | 完全自定义，兼容性好       |

# 8. 平台适配

## 8.1 移动端（Android/iOS）

```
限制：
  - GPU Fill Rate 有限（减少 Overdraw）
  - 内存带宽有限（减少贴图采样次数）
  - 不支持 Geometry Shader（部分设备）
  - 不支持 Tessellation

适配方案：
  皮肤：
    ✅ 厚度贴图 SSS（代替屏幕空间模糊）
    ✅ Blinn-Phong 代替 GGX（减少 ALU）
    ❌ 关闭法线模糊
  头发：
    ✅ 双 Pass（Depth Prepass + Alpha Blend）
    ✅ Early Z 优化
    ❌ 减少飞丝层（最多 2 层）
  眼睛：
    ✅ Parallax 折射近似
    ❌ 关闭环境反射（或使用极低分辨率 Cubemap）
  描边：
    ✅ 顶点描边（代替屏幕空间后处理）
  毛发：
    ✅ 8 Shell 层（代替 16+）
    ❌ 关闭 Fin
```

## 8.2 PC/主机

```
特性支持：
  ✅ 完整 GGX BRDF
  ✅ 屏幕空间 SSS
  ✅ 环境反射（高分辨率 Cubemap）
  ✅ Geometry Shader（毛发 Shell 合并）
  ✅ Tessellation（皮肤细分）
  ✅ 屏幕空间描边（后处理）
  ✅ 16~32 Shell 毛发层
```

## 8.3 WebGL / 微信小游戏

```
限制：
  - 不支持 Geometry Shader
  - 不支持 MRT（多渲染目标）
  - 贴图格式限制（不支持 BC 压缩，使用 ASTC/ETC2）
  - 内存极度受限

适配方案：
  - 关闭毛发（Shell Fur）
  - 关闭屏幕空间描边
  - 皮肤使用最简 Lambert + 简单高光
  - 头发使用单 Pass Alpha Test
  - 贴图降至 512x512
```

## 8.4 质量分级配置

```csharp
// 根据设备性能自动选择质量等级
public enum CharacterQuality { Low, Medium, High, Ultra }

public static CharacterQuality GetQuality() {
    int memoryMB = SystemInfo.systemMemorySize;
    int gpuMemoryMB = SystemInfo.graphicsMemorySize;

    if (memoryMB >= 8192 && gpuMemoryMB >= 4096)
        return CharacterQuality.Ultra;   // PC/主机
    else if (memoryMB >= 4096)
        return CharacterQuality.High;    // 高端手机
    else if (memoryMB >= 2048)
        return CharacterQuality.Medium;  // 中端手机
    else
        return CharacterQuality.Low;     // 低端手机
}

// 根据质量等级设置 Shader 关键字
void ApplyQuality(Material mat, CharacterQuality quality) {
    mat.DisableKeyword("_QUALITY_HIGH");
    mat.DisableKeyword("_QUALITY_MEDIUM");
    mat.DisableKeyword("_QUALITY_LOW");

    switch (quality) {
        case CharacterQuality.Ultra:
        case CharacterQuality.High:
            mat.EnableKeyword("_QUALITY_HIGH");
            break;
        case CharacterQuality.Medium:
            mat.EnableKeyword("_QUALITY_MEDIUM");
            break;
        case CharacterQuality.Low:
            mat.EnableKeyword("_QUALITY_LOW");
            break;
    }
}
```
